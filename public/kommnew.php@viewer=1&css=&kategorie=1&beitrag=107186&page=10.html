Dieser Auszug aus dem Artikel zum Thema Exploit aus der Wikipedia sagt eigentlich alles Wesentliche:

Ein Exploit (englisch to exploit - ausnutzen) ist ein Computerprogramm oder sgript, welches spezifische Schwächen beziehungsweise Fehlfunktionen eines anderen Computerprogramms zur Erlangung von Privilegien oder in Absicht einer DoS-Attacke ausnutzt.

Ein Exploit wird oft auch einfach nur zur Demonstration einer Sicherheitslücke geschrieben und veröffentlicht. Dieser Beweis der Machbarkeit wird in der Fachsprache auch Proof of Concept genannt. Dadurch soll erreicht werden, dass Hersteller von Software möglichst schnell auf bekannt gewordene Sicherheitslücken reagieren. Oft bezeichnet man auch nur die theoretische Beschreibung eines Exploits als Exploit.

Meist machen sich Exploits zu Nutze, dass Computer nicht zwischen Programmcode und Daten unterscheiden. So wird z.B. bei einem Pufferüberlauf der Code des Angreifers in einen nicht dafür vorgesehenen Speicherbereich geschrieben, wodurch die Ausführung der Anwendung manipuliert werden kann. Ein Angreifer kann oft eigenen Code zur Ausführung bringen, der ihm beispielsweise eine Shell mit den Privilegien der ausgenutzten Anwendung liefert.

Dieser Artikel basiert auf dem Artikel Exploit aus der freien Enzyklopädie Wikipedia und steht unter der GNU-Lizenz für freie Dokumentation. In der Wikipedia ist eine Liste der Autoren verfügbar.

Aus der Ankündigung der Spielleitung
"Nötig macht diese Änderung leider der Exploit des Verfallsystems von einigen Wenigen. "

Der Begriff Exploit ist hier im eigentlichen Sinne tatsächlich falsch gewählt.

Auszug aus Wikipedia
"Ein Exploit (englisch to exploit - ausnutzen) ist ein Stück Software oder eine Sequenz von Befehlen, welches spezifische Schwächen beziehungsweise Fehlfunktionen eines anderen Computerprogramms zur Erlangung von Privilegien oder in Absicht einer DoS-Attacke ausnutzt."

Aus den Nutzungsbedingungen
"Die vorsätzliche Nutzung einer Programm-Schwäche (Exploit) oder Fehler (Bug) ist verboten. "

In anbetracht der Tatsache, dass es weder Strafen für die Spieler, die den "Exploit" exzessiv nutzen, noch eine sofortige Änderung gab, lässt nur den Schluss zu, dass der Begriff "Exploit" im übertragenen Sinne verwendet wurde.
Bleibt also festzustellen, dass kein Verstoß gegen die Spielregeln vorliegt und sich keiner der Betroffenen in irgendeiner Form rechtfertigen müsste.

Im auf unseren Fall übertragenen Sinne, ist ein Exploit eine konzeptionelle Schwäche. Derjenige der diesen Exploit nutzt verschafft dadurch in der Regel einen Vorteil gegenüber denjenigen die ihn nicht nutzen.
