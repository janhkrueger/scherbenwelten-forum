Durmundo (RIP) So ist es momentan:
Werden in einer Stadt Armeen ausgehoben bekommt der Bürgermeister keine Nachricht.
So soll es werden:
Wird eine Armee in einer Stadt ausgehoben, bekommt der Bürgermeister eine Diplo in der steht, wieviele Armeen ausgehoben wurden und wer dies gemacht hat.
Vorteile für das Spiel:
Der Bürgermeister weiß wieviele und wer Armeen ausgehoben hat.
Nachteile für das Spiel:
Bürgermeister können schwerer überrascht werden.
Mögliche Problematiken:
keine

Eine Aufteilung der Freundschaftserklärung in Turmdurchlass, Stadt- und Routenbenutzung bei zuviel Bösartigkeit, Wachdienste/Raubüberfälle, Benutzung von Gasthäusern und Armeeaushebung würde diesen Vorschlag natürlich nichtig machen. ;) 		07.11.08 20:15
