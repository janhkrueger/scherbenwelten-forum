Koveras Alvaron (RIP) So ist es momentan:
Ein Wachdienst stellt die (zeitversetzte) Möglichkeit dar, auf eine bestimmte Situation zu reagieren, von der man annimmt, dass sie bald eintritt - und das ohne dass man online sein muss.
Es gibt 3 Arten von Wachdiensten (Wachdienst, Raubüberfall, Hinterhalt). Ausgelöst werden sie allesamt beim Betreten des Feldes.
So soll es werden:
Zu den bestehenden 3 Wachdiensten soll ein 4. hinzugefügt werden (Namensvorschlag "Bewachen"). Ausgelöst wird dieser wenn genau das Gebäude, welches man bewacht bestohlen wird.
Vorteile für das Spiel:
Eine sinnvolle Verteidigungsmöglichkeit gegen Diebe, welche Diebe nicht unnötig schwächt aber die verhindert, dass ein spezielles Gebäude überdurchschnittlich oft bestohlen wird (insb. von ein und derselben Person).
Nachteile für das Spiel:
Diebe haben es schwerer, insbesondere wenn sie eine bestimmte Person "fertig" machen wollen.
