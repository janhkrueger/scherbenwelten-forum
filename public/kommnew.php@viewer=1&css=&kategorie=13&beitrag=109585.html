System/Engine/Technisches - Nation auflösen (1 )
Valera von Avatraz (RIP) So ist es momentan:
Um eine Nation auslösen müssen alle Mitglieder die Nation verlassen oder rausgeworfen werden. Der Nationsführer muss sich selbst rauswerfen, bleibt abr Nationsführer obwohl er kein Mitglied ist bis zur nächsten Hauptrunde.
So soll es werden:
Im Nationsbild des Nationsführeres gibt es irgendwo ein hübsches "Nation auflösen" mit Sicherheitsabfrage, das generiert auch einen schönen Historyeintrag ala "Die Nation Blubbeldiblubb wurde aufgelöst" alle Mitglieder erhalten eine Nachricht und sind von diesem Moment an Nationslos, auch der Nationsführer selbst.

Die Kosten hierfür sollten ebenfalls an 1k AP gekoppelt werden da es so in etwa das selbe entspricht wenn man sich selbst kicken muss.
Vorteile für das Spiel:
Keine Verwirrenden Historyeinträge von wegen Nationsführer wurde aus der Nation geworfen.
Neulinge (oder auch ältere Spieler) die eine Nationgründeten, aber entscheiden einer anderen Nation beizutretten ist es ersichtlicher wie man eine Nation auflöst, bzw. als Anführer verlässt.
Nachteile für das Spiel:
Irgendwelche DAUs und Acchacker werden Nationen einfacher ungewollt auflösen. Aber ich hoffe Hacking ist kein Argument dafür.
Mögliche Problematiken:
seh ich keine
