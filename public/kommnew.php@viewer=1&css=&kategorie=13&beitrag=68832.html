Marduc (RIP) So ist es momentan:
Befindet sich ein Seefahrer in der Gruppe, so können auch andere Mitglieder der Gruppe in Seefahr geschult werden. Ziel ist es meist, den etwas schwachen Seefahrer durch z.B. einem Jäger zu ersetzen. Wird der Seefahrer aus der Gruppe entlassen, so zählt der beste Seefahrerskill eines Gruppenmitgliedes. Sonst gibt es keinerlei Einschränkungen.
So soll es werden:
Gruppen ohne Seefahrerchar können nur innerhalb der Inselregion zur See fahren. Wenn das Schiff den Regionsrand erreicht, sollte das Schiff nicht weiterfahren können (der Kapitän weigert sich auf offene Meer hinaus zu segeln). Befindet sich ein Seefahrerchar in der Gruppe, so kann das Schiff die Inselregion auch verlassen. (Mit Region meine ich den Bereich der noch vom Regionschat betroffen ist)
Vorteile für das Spiel:
Der Charakter Seefahrer bekommt eine stärkere Bedeutung.
Nachteile für das Spiel:
Viele Laufgruppen müssten neue überdacht werden.
Mögliche Problematiken:
Wenn der Regionsrand erreicht wird, könnte eine Rückfahrt evtl. nicht mehr ausreichen.
