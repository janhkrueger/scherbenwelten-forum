Vincent Aquila (RIP) So ist es momentan:
Bisher ist es so, dass Änderungen umgesetzt werden, obwohl sich nur ein Bruchteil der Spielergemeinschaft zu dem Thema geäußert hat.
So soll es werden:
Änderungen sollten nur durchgeführt werden, wenn sich mindestens 50% der Spielergemeinschaft dazu geäußert haben. Das können Sie ja über den Smilie bekanntgeben.
Vorteile für das Spiel:
Änderungen werden nur dann umgesetzt, wenn wirklich die Mehrzahl der Spieler davon Kenntnis genommen haben.
Nachteile für das Spiel:
Es gibt keine Änderungen mehr, von dennen Spieler überrscht werden, die sich zu dem Thema noch nicht geäußert haben.
Mögliche Problematiken:
Keine Änderungen mehr Aufgrund einer kleinen Anzahl von Spieler, die sich hier einbringen um Ihre Interessen durchzusetzen.
Sonstiges:
