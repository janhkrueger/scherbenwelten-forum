Forum/Diplo/Chat - Diplo bei Handelsroutenentfernung (7 )
Balëbréthilion Târmundor (RIP) So ist es momentan:
Wenn eine Route von Stadt A nach Stadt B entfernt wird, erhält nur Bürgermeister A eine Nachricht. ("Routenleger hat eine Handelsroute von Stadt A entfernt!")
So soll es werden:
Wenn eine Route von Stadt A nach Stadt B entfernt wird, erhalten Bürgermeister von A und B eine Nachricht. ("Routenleger hat eine Handelsroute von Stadt A entfernt!" und "Routenleger hat eine Handelsroute nach Stadt B entfernt!")
Vorteile für das Spiel:
Routen zu einer Stadt hin haben meiner Ansicht nach gleichwertige Bedeutung für die Stadt und den Bürgermeister wie Routen die von der Stadt zu einer anderen Stadt hin führen.
Auch ist für beide Arten von Route eine Verwaltung im Stadtmenü möglich - für beide eine Diplo einzuführen wäre also nur konsequent.

Beim Anlegen neuer Routen erhält der Bürgermeister ja auch eine Nachricht für beide Typen von Route und nicht nur für Routen die von der Stadt aus weg führen.
