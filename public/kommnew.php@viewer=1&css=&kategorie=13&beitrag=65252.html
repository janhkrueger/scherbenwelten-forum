Neues Gebäude: Werkstatt
Landstreicher (RIP) Idee:
In einer Werkstatt sollen zusätzlich zu normalen Produktionsgebäuden Rezeptgegenstände produziert werden können.

Anforderungen:
# Ein Experte (NC/HC mit entsprechendem Skill) muss in der Werkstatt sein,

# Das entsprechende Rezept muss vorhanden sein
(ggf. könnte man sagen je Gebäude 1 Rezept, wobei das für die Werkstatt nur eine Abschrift sein kann - Kostenfaktor vielleicht die Hälfte vom Original)


Nutzen:
Steigerung der Produktion.

Sonstiges/Anmerkung:
Natürlich kann man nun sagen, "dann bau dir halt n 2ten Steinbruch, stell dein Schürfer rein und mach in 2en Anbauten"
Die Idee mit der Werkstatt soll aber die sein, das es kein spezielles Gebäude für nur eine Art von Rezept sein soll.
Jeder Arbeiter kann darin arbeiten, jedes Rezept verwendet werden.

Je nach Ausbau wäre ggf. möglich 2, 3, mehr, verschiedene Rezepte gleichzeitig zu produzieren.
