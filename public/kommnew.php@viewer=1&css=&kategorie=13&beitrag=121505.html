RPG - Heirat/Verlobung -> Zustimmen (65 )
Iga Pelogran (RIP) So ist es momentan:
Drücke ich jemanden einen Ring in die Hand kann ich Ihn heiraten, ohne dass er mich heiraten muss oder zustimmen. Eine evtl. bestehende Ehe wird dadurch annuliert.
Beispiel: Falster ist mit Titania verheiratet. Treffe ich Falster, drücke ihm den Ring in die Hand und Klicke auf Heiraten, ist er mein Mann und von Titania geschieden.
So soll es werden:
Wenn jemand sich verlobt oder heiratet, muss dass Ziel das Zustimmen!
Vorteile für das Spiel:
- Sinn?
- Kostenersparnis, da man nicht x mal den gleichen heiraten muss, weil irgendjemand anderes was dagegen hat.
- Keine Zerstörmöglichkeit von RP
Nachteile für das Spiel:
- Das RP anderer Leute kann nicht mehr behindert werden, auf diese Weise
Mögliche Problematiken:
Programmierung / Art der Zustimmung
