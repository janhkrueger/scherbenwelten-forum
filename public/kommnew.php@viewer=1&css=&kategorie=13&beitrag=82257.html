_Sonstiges - Reiserouten auf Nation/Spieler eingrenze
Varrok von Rabensturm (RIP) So ist es momentan:
Die Handelsrouten können nur durch den BM für einzelne Spieler verboten werden.
So soll es werden:
Es sollte so werden, dass der BM jede exitierende Handelsroute individuell auf Nationen/Spieler freigeben oder verbieten kann.
Vorteile für das Spiel:
Somit können zum Beispiel Nationen die Handelsrouten unter den eigenen Städten, z.B. nur für interne Mitglieder, freigeben. Ein weiterer Vorteil ist, dass so andere Spieler nicht von außerhalb eindringen, aber die freigegebenen trotzdem nutzen können. Dadurch muss nicht extra auf eine Route verzichtet werden und man spart obendrein vielleicht noch AP, da man ja sonst eventuell durch laufen mehr verbrauchen würde.
Nachteile für das Spiel:
Keine bekannten...
aber dafür seid ihr ja da :)
