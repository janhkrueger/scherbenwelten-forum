Fabius Alagos
So ist es momentan:
Man bekommt XPs durchs Erlegen von Monstern oder durch das Ernten von Rohstoffen.
Für einen "erfahrenen" Spieler ist irgendwann nur noch die XP-Erlangung über das Sammeln möglich, da die Monster keine oder kaum XPs mehr bringen.

So soll es werden:
Andere Tätigkeiten bringen auch oder stattdessen XPs. Zum Beispiel Kills von Monstern die man "zum ersten Mal" erledigt. Das wäre dann nicht nur für die Galerie sondern wirklich eine "neue" Erfahrung und eine gezielte Suche nach neuen Monstern würde spannender. Ich denke aber auch an andere Erfahrungen wie: Das erste Errichten einer neuen Gebäudeart, Das erste Übernehmen eines Bürgermeisteramtes abhängig von der Größe der Stadt, das Erfüllen von Aufträgen, das erste Erstellen von einem neuen Gegenstand abhängig von den Rezeptkosten, das Besiegen anderer Spieler im PvP abhängig von deren Stellung in der Ruhmstatistik, das Abbrennen von Gebäuden abhängig von deren Großausbauten, das Abbrennen von Städten abhängig von deren Größe, das Aufreiben einer Armee abhängig von deren Größe, vielleicht noch irgendwas was mit Glauben und Konvent in Verbindung steht, das Erste Öffnen einer Truhe, der Einbruch in Gebäude abhängig von deren Struktur, der erste Diebstahl von Gegenständen abhängig von deren Diebstahlsfaktor, das erste Betreten einer Stadt/Region in der man noch nie war usw.

Vorteile für das Spiel:
Die langatmigen Erntephasen würden möglicherweise durch aktiveres Spiel ersetzt.

Nachteile für das Spiel:
Nicht absolute Chancengleichheit für alle Berufe. Das war allerdings noch nie so.

Mögliche Problematiken:
Ich bin mir dessen bewußt, dass dies eine relativ einschneidende Änderung wäre. Da keine neue Runde geplant ist, möchte ich den Vorschlag trotzdem noch bringen, da es aus meiner Sicht die einzige wirkliche "Schwachstelle" des Spiels ist.
