_Sonstiges - Reiserouten freischalten (7 )
Elder von Adlerfeste (RIP) So ist es momentan:
Sobald eine Reiseroute gelegt ist und Pferde eingelagert wurden kann man diese benutzen.
So soll es werden:
Die Reiseroute muss vom Bürgermeister der Zielstadt freigeschaltet werden bevor man sie benutzen kann.
Vorteile für das Spiel:
- Ein einzelner Routenleger mit Passierschein kann keine Festung mehr aushebeln.

- Festungen funktionieren wie gedacht, müssen also i.A. frontal angegriffen werden.
Nachteile für das Spiel:
- siehe Vorteile

- Wenn der Bürgermeister abwesend ist können Routen zu Anfangs nicht genutzt werden.
Mögliche Problematiken:
keine ersichtlich
