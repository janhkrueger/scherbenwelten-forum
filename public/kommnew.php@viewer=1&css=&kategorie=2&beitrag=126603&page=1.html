Spielregeln

Es gibt insgesamt 30 Einsatzpunkte, die man je auf eine Viertelstunde (15 Minuten) verteilen kann.

- Bei einer Differenz von 0-1 fällt kein Tor.
- Bei einer Differenz von 2-4 Toren fällt ein Tor für die stärkere Mannschaft.
- Bei einer Differenz >=5 fallen zwei Tore für die stärkere Mannschaft.

Beispiel:

Mannschaft A: 5-4-5-6-4-6
Mannschaft B: 1-1-10-5-5-8

Nach Spielminute // A:B
15 // 1:0
30 // 2:0
45 // 2:2
60 // 2:2
75 // 2:2
90 // 2:3
