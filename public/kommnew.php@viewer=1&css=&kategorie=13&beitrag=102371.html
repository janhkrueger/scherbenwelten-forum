Coran (RIP) So ist es momentan:
Jeder der aus einer Nation mit Nat KE´s geht oder gegangen wird erhält von allen Mitgliedern der entsprechend verfeindeten Nationen eine Privat KE.

Unabhängig von Freundschaften etc.
So soll es werden:
Es bleibt so ABER jeder der Natmitglieder bekommt eine Nachricht dass er eine Privat KE gegen XYZ ausgesprochen hat. ODER
Jeder Spieler bekommt beim nächsten Log In ein Pop Up in dem er entscheiden muss ob er eine KE ausspricht oder nicht. (das wäre dann auch kostenlos falls ja)
Bis dieses Pop Up beantwortet würde würde der entsprechende Spieler wie mit einer Privat KE belegt behandelt.
Vorteile für das Spiel:
Feindeslisten werden nie kontrolliert, vor allem da sie immer länger werden macht sich niemand die Mühe wirklich zu überprüfen ob die entsprechenden KE´s so noch sinnvoll sind.
Die Feindeslistenflut ist also eigentlich eher ein Ausdruck der Faulheit der Spieler als wirklich eine bewusste Feindesliste.

Auf die vorgeschlagene Art und Weise ist der Ausspruch einer Privat KE wieder etwas persönliches, bewusstes und gewachsene Strukturen können gewahrt bleiben ohne das wieder massig AP und Anschreibzeit in Freundeserklärungen etc gesteckt werden müssten.
Nachteile für das Spiel:
Keine mir bekannten.
Klar ein Fall von ich will es einfacher ist es schon, aber ich sehe Privat KE´s nun einmal als etwas persönliches an das man als Spieler bewusst auslösen sollte.
Mögliche Problematiken:
Programieraufwand, vor allem sollte die Pop UP Funktion genutzt werden.
Sonstiges:
Kampf dem KE Spam !
