Forum/Diplo/Chat - Gasthausnutzung
Der schwarze Ritter (RIP) So ist es momentan:
Wenn ein Gasthaus leer ist kann es nur vom Besitzer aufgefüllt werden. Das heißt man steht öffters mal vor leeren Gasthäusern.
So soll es werden:
Es werden Elemente von der Burg und vom HP implementiert. So kann auch ein Kunde das Gasthaus auffüllen, wie bei der Burg, Abrechnung erfolgt dann wie beim HP. Außerdem wäre es noch ganz schön, wenn der Zustand der Auffüllung automatisch angezeigt werden würde. So daß man gleich vorher weis, welche Waren man auffüllen sollte umes benutzen zu können, bzw. was eben fehlt.
Vorteile für das Spiel:
Es würden nicht so viele nutzlose leere Gasthäuser herum stehen.
