Freya san Gordar (RIP) Ich hab mal Grafiken für die Innenstadtpflaster in der Normalansicht gebastelt. Dabei ist mir aufgefallen, dass es bei dem derzeitigen System mit der Beeinflussung der Nachbarfelder öfter zu unschönen Anzeigefeldern kommt. Daher hab ich mir gleich auch mal ein neues System ausgedacht. Dazu ist leider auch ein neues Indizierungssystem nötig. Die alten Nummern können mit einer Indexabbildung behalten werden.



Das Nummernsystem

Die automatisch generierten Felder sind 52**, die manuell gesetzten 53**, wie bisher.
Ich hab die Ecken binär gewichtet. Rechts unten 1, Links unten 2, Rechts oben 4, links oben 8. Der Index ergibt sich aus der (gewichteten) Summe der gepflasterten Ecken.
zB ist 5207 (1+2+4).
Die Wegegrafiken (53**) haben willkürliche Numerierungen, lediglich die Weganfänge haben den gleichen Index wie die entsprechenden 52**-Grafiken.



Feld pflastern
das aktuelle Feld wird auf 5215 gesetzt. Alle Nachbarfelder, die bisher den Index 0 haben (0.gif), werden (provisorisch auf) den Index 5200 gesetzt. 53**-Felder werden nicht verändert.
