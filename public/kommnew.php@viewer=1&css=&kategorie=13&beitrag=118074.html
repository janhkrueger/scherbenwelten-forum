Iga Pelogran (RIP) Wie wärs einfach:
Bei solchen ereignissen:
1. Sobald der Server läuft
-> Log-In Seiten Hinweis
-> Kurzer Dreizeiler ins Forum von off. Seite
-> oder aber eine Quest (wie blauer Trank) nur mit dem Hinweis und als Option "nichts geben"

2. Unter Stellung / Register Optionen hinter EMail-Adresse eine Checkbox. Bei Störung EMail an mich (oder so ähnlich).
Dann können Wissbegierige Ihren Info Durst selber regeln und sind auch bei Server-tot erreichbar und informiert.
