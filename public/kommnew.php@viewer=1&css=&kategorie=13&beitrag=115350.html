Emilie de Witt (RIP) So ist es momentan:
Man muss für das Fischen, immer hinaus auf die See fahren, oder in einem kleinem Tümpel mit einem Fischerboot fischen.
Recht unkompliziert und alles
So soll es werden:
Man könnte eine Fischerhütte bauen, so wie bei Anno 1602 zB, an Küstenfelder, zur Offenen See hin, man könnte dann die Seeleute die man nicht mehr zwingend braucht, weil man einem anderem Charakter genug Seefahrt beigebracht hat, weiter verwenden, und unkompliziert Fische produzieren :)
Vorteile für das Spiel:
Naja, Vorteile nicht zwingend, unkomplizierter halt, allerdings, denke ich schon, das es auch in der damaligen Zeit Fischerhütten gab ;)
es wäre nur eine anpassung an die historie
