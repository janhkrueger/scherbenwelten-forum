Liste Änderungen Vertretungsregelung (8 )
Support
Wie bereits hier http://www.scherbenwelten.de/kommnew.php?viewer=&css=&kategorie=1&beitrag=115939
angekündigt, wurde mit heutigem Abend eine Änderung in den Vertretungsoptionen online gestellt.
Hier eine Liste aller Einschränkungen für Vertretungsaccounts:

- Kein Zugriff auf die Optionen
- Keine Gebäude abreissen
- Kein Löschen von NPCs
- Kein Zugriff aufs reguläre Forum
- vorerst kein Konventszugriff
- Keine Erweiterung der Liste der Vertretungsberechtigten
- Kein Zugriff auf das RPG-Menü (sofern vorhanden)
- Diplomatie: nur Erteilung von Kriegserklärungen möglich
- Kein Plündern von Gebäuden
- Keine Zerstörung von Gebäuden
- Kein Bestehlen von Gebäuden
- Keine Hinterhalte
- Raubüberfälle können nur auf eigenen Gebäuden gelegt werden
- Wachdienste auf allen Nationsgebäuden
- Keine Stadtübergabe
- Nationsführermenü ist uneingeschränkt vorhanden
- Turmeinstellungen können nicht verändert werden
