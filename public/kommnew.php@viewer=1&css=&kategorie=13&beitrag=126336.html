Skills/Berufe/Rassen - Völlig sinnfreie Berufe
Iga Pelogran (RIP) So ist es momentan:
Es gibt einige Berufe, die man als HC oder seinen NCs lehren kann. Die hierfür benötigten Kenntnisse werden geschult.
So soll es werden:
Es sollen neue Berufe erstellt werden (oder ein neuer) der wie folgt ist.

Der Beruf wird bei nahezu allen Spielern gebraucht.
Der Beruf kann erlernt werden, jedoch kann der Char nicht anderweitig geskillt werden (alle Schulen, alle Fächer von Basis 0 auf 1 kostet 5001 AP)
Der Beruf kann eine bestimmte Form von Aktion durchführen, die nur er kann. (Vergleich Heiler->Alle heilen, Behandeln)
Bitte Beispiel beachten.
Vorteile für das Spiel:
Der Beruf wird von nur wenigen Spielern, meist Neulingen gewählt, denn die erfahrenen Spieler haben kein Platz weder im Prodgebäude noch im Tempel um diesen sonst Nutzlosen CHar zu parken. Dafür geben sie aber einige Goldstücke aus, und helfen somit den Spielern, die sich einen solchen Char "leisten".
Nachteile für das Spiel:
Wieder mehr Komplexität.
Mögliche Problematiken:
Programmieraufwand
Sonstiges:
Hier ein Beispiel:
Beruf Schornsteinfeger.
Der Schornsteinfeger hat keine Skills (alles Basis 0) und kann auch nix lernen. Er kann auch nur Straßenkleidung tragen.

In jedem Prodgebäude muss 1x pro Jahreszeit der Schornstein gefegt werden, da ansonsten das Gebäude in Flammen aufgeht (Ein Teil der Ausbauten (max. 1GA) verfallen) Dies gilt für alle Außer-Stadt-Gebäude. Hierzu gibts einen Schalter "Schornstein fegen (20 AP)". Dies kann jedoch nur vom Feger erledigt werden. Eine Anzeige zeigt an, ob im Gebäude schon gefegt wurde oder nicht.

So werden sich die erfahrenen Spieler nicht so einen Stein ans Bein binden und lieber für die Dienstleistung zahlen, sodass neuere Spieler damit eine zusätzliche Einnahmequelle finden können.
