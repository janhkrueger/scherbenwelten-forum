Kampf/Armeen/Monster - Kopfgeldjägervertrauen
Radalitz Trollblut
So ist es momentan:
Durch das Kopfgeld kann man als Krieger nicht leben.
Die Kopfgeldjägeraufträge bringen pro erlegtes Monster von 1 - ca. 400 Goldstücke ein, je nachdem ob sie eine Bedrohung für eine Stadt darstellen das doppelte an Goldstücken.
Bei ausreichend Monstern kann man vielleicht 40-50.000 Goldstücke am Tag (normale Tages-AP) erwirtschaften, abzüglich der Gasthauskosten, die vielleicht anfallen.
Solche Summen decken bei weitem nicht die Anschaffungskosten der Ausrüstung, die darüber zerstört werden, wenn man es kontinuierlich macht.
Welcher Krieger würde schon diesen Beruf ergreifen, wenn er dabei noch drauf zahlen muss....da geht er doch lieber Äcker pflügen.
So soll es werden:
Abhängig vom Einzelskill/Gruppenskill (z.B. Handeln) kann das Kopfgeld erhöht werden. Z.B auf das 5-10 Fache.
Vorteile für das Spiel:
Das Kopfgeld wird realistischer berechnet, sodass man sich zumindest seine Ausrüstung/Gasthaus-Kosten leisten kann.
Man kann endlich ungebunden Monster jagen gehen.
Nachteile für das Spiel:
Unter Umständen mehr Monster-Jäger, die keine Gebäude mehr betreiben, da Sie nicht mehr darauf angewiesen sind.
Mögliche Problematiken:
Mir keine bekannt.
Sonstiges:
Es gibt sicherlich noch andere Sinnvolle Möglichkeiten das Kopfgeld attraktiver zu gestalten, genauso gibt es noch weitere Vorschläge zu diesem Thema, nur wurden diese bisher nie von einem Skill abhängig gemacht. Handeln ist bislang kein wirklich wichtiger Skill, vielleicht wird er es dadurch ja. 		29.05.14 13:12
