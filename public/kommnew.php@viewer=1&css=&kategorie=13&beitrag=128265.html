Gebäude/Rezepte/Produktion - Absolut Nutzloses (1 )
Jolina (RIP) So ist es momentan:
Die neue Kategorie kann leider nur im Magierturm hergestellt werden.
So soll es werden:
Diese Kategorie sollte von allen Gebäuden herstellbar sein.
Vorteile für das Spiel:
Jeder hat etwas davon, und muss nicht erst einen Magierturm errichten. Der Char kann ja trotzdem Alchemie als Skill benötigen, um das Rezept zu produzieren.
Nachteile für das Spiel:
Eine Schwemme von absolut nutzlosen Gegenständen. - Die gab es eigentlich auch schon bisher. =D
Mögliche Problematiken:
1. Während der Produktion des Gegenstandes können dadurch auch weiter Waren produziert werden, im Gegensatz zu einem Char im Magierturm.

mögl. Lösung: Dem kann man entgegenwirken, in dem man solange der Gegenstand als Produkt eingestellt wird den Char mit dem höchsten Alchemieskill für die Produktmengenberechnung heraus nimmt.

2. In einer Schmiede könnte man solche Gegenstände mit Schichten Aufwerten.

mögl. Lösung: Der Schichtenmodus wird bei dieser Kategorie einfach ignoriert.
