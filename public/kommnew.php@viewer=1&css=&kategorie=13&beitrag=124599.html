Ossian Montfort of Éire Castle (RIP) So ist es momentan:
In der Handelsgilde wird der Ankauf einer Ware/ Warenmenge in einer Stadt angezeigt, obwohl selbige nicht genug geld zum Ankauf hat.
So soll es werden:
In der Handelsgilde wird nur die max. Verkaufbare Menge auf Basis des eingelagerten Geldes angezeigt.
Vorteile für das Spiel:
Weniger Frust für die Händler - wenn ich schon die Info bekomme, das in einer tadt eine Ware gesucht wird, dann sollte auch automatischj mitgeteilt werden, was ich max. verkaufen kann bei vorhandenem geldbeutel.
Nachteile für das Spiel:
- die Spieler die nörgeln, man will es "einfacher" obwohl es hier nur um eine Klarstellung von Informationen geht
Mögliche Problematiken:
- die Liebe Programmierung
