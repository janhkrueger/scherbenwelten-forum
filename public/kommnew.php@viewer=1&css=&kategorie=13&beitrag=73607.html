So ist es momentan

Eine relativ kleine gruppe spieler kann einen Konvent übernehmen sofern die Opferwaren vorhanden sind.

So soll es werden

Hohepriester können nicht einfach zum Propheten aufsteigen, sondern müssen zuerst 2 vorgänge durchmachen:

1 sie müssen genügend opfern.
2 sie müssen von den konventsmitglieder mit einer mehrheit gewählt werden (jeder hat nur 1 stimme)


Vorteile für das Spiel

Es fördert das RPG da die Konvente nicht so leicht beeinflusst werden können und hohepriester müssen sich erst durch taten auszeichnen damit sie aufsteigen können im konvent (hoffe wenigstens das es diesen effect hätte)


Nachteile für das Spiel

sehe ich momentan keine

Mögliche Problematiken

In kleineren Konventen könnte der Effect umgedreht werden.

5 spieler sind im konvent und 3 wollen die macht in diesem. Die 3 ernennen einen hohepriester zum prophet und die 2 übrigen spieler können nichts Unternehmen
