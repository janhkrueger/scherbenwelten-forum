Chevonne Siobhan (RIP) So ist es momentan:
Derzeit gibt es zwei Arten von Foren: Das RPG-Forum und das Nationsforum.
So soll es werden:
Ich wünsche mir eine dritte Art: Die Geheimgesellschaft.

Was soll an diesem Forum anders sein?

Die Geheimgesellschaft zeichnet sich dadurch aus, dass die Mitglieder wie bei einem Absiebenkonvent ein Pseudonym verwenden können.

Natürlich muss das auch irgendwie in der Engine einen Unterschied machen. Im Gegensatz zu den anderen Arten müssen für jedes Mitglied zwei Sitze erstellt werden. Einer für den richtigen Namen, einer für das Pseudonym. Das Pseudonym kann nur im selben Forum verwandt werden.
Vorteile für das Spiel:
Das RPG wird um eine Variante reicher. Zudem werden die Chars bei eventueller Unterwanderung geschützt. Der Forengründer kennt natürlich alle seine Mitglieder. Aber er wird ja auch kaum sein eigenes Forum unterwandern.
Nachteile für das Spiel:
Sehe ich grad nicht. Aber das ist ja immer so. *g*

Vom Programmieraufwand abgesehen, sehe ich keine.
