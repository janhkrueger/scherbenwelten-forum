Iga Pelogran (RIP) So um mal die Windows/Anti-Windows Diskussion zu beenden:

Windows ist ein hervorragendes Betriebssystem welches für den Endanwender, den User, geschrieben wurde. Der größte Teil der Hard- und Software (im Anwenderereich) wird für Windows hergestellt. Vor allem für Leute die gerne Spielen ist Windows die erste Wahl. Die stabilität hat sich in den letzten Jahren wirklich verbessert, jedoch bedarf die Sicherheit ständigen Updates. Für den User ist das Betriebsystem leicht zu erlernen und zu bedienen. Linux ist für erfahrene User und Admins besser geeignet. Das Betreibssystem nutzt nich soviele Resourcen wie Windows, was das schnellere Arbeiten gewährleistet da für die Programme mehr Resourcen zu Verfügung stehen. Nahezu jede Hardware wird (das richtige Linux vorausgesetzt) Standardmäßig erkannt und mit einem Standardtreiber betrieben, viele Hersteller liefern jedoch eigene Linuxgtreiber zur Beschleunigung dieser mit. Der weitaus größere Vorteil bei Linux ist die Freiheit, die man als Systemadmin genießt, gerade im Serverbereich. Benutzereinstellungen, Rechtevergaben usw ist mit Linux einfacher und sicherer. Zur Sicherheit sprechen die Anzahl der Viren, die es auf den Systemen gibt. Für Windows sind hunderttausende unterwegs, für Linux eine Hand voll. Ein noch größerer Vorteil von Linux ist der Preis. Nur wenn man Support und ein gedrucktes Handbuch will, muss man die boxed Version kaufen. Ansonsten legal und FREI herunterladbar. Das Frei bezieht sich auch auf GPL, die es ermöglicht Programmieren sämtliche Software auf Ihre Bedürfnisse zurechtzuschneiden, da bei nahezu aller Software der Quell(Programmier)code mitgeliefert wird...
