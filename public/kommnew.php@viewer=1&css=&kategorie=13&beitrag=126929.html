Skills/Berufe/Rassen - Intelligenz Ernten
Iga Pelogran (RIP) So ist es momentan:
Ist ein Erntetrupp über einen bestimmten Skill, so erntet er alles auf einem Feld bis auf 0 runter. Das ist effekttiv aber nicht intelligent, da die Felder, die auf 0 sind, nicht so schnell nachwachsen, wie die, die noch etwas drauf haben. Ein Ungelernter Trupp kann auf einem 3er Getreidefeld stehen und hunderte male ernten und dadurch Hunderte oder gar Tausende Getreide Ballen ernten, da sie die Feldstärke nicht verändern, ein gelernter Trupp bekommt aus dem gleichen Feld maximal 3 Getreide.
So soll es werden:
Ausgebildete Erntetrupps sollten soweit Intelligent sein, dass man sie steuern kann, ganz abernten (Feld bis auf 0 runterbringen) oder bis zu einem "Mindestbestand" abernten (evtl. bis auf 10 runter).
Natürlich wäre das schwer zu programmieren und ein zweiter Button wäre doof. Deshalb würde ich sagen, das maximal 90-95% geerntet wird (Aufgerundet) Das wären bei einem 100er Feld 95, bleiben 5, 2. ernten 4,75 aufgerundet 5, bleibt 0.
Natürlich sollte das für alle Massenwaren gelten.
Vorteile für das Spiel:
- gute Ernter würden nicht blind ernten sondern intelligent
- Abgeerntete Felder regeneriern sich schneller
- Steuerbarkeit, ob ein Feld komplett abgeerntet werden soll
Nachteile für das Spiel:
- Man muss mehrmals ernten um ein Feld wirklich komplett abzuernten.
Mögliche Problematiken:
- Programmierung
