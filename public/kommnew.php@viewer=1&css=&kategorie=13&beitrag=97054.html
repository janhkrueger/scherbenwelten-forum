_Sonstiges - Freundschaft (1 )
Georg (RIP) So ist es momentan:
Es gibt eigentlich nur drei Zustände:
Freundschaft, Neutal, Feindschaft
Leider ist es aufgrund von Sicherheitsaspekten manchmal notwenidig, Freundschaften zu beenden, obwohl dies keine Aussage über die eigentlichen Zustände sind.
So soll es werden:
Entweder
könnte man sich einen Zwischenzustand "Sympathie" vorstellen
oder
es ist denkbar, dass die Details der aus einer Freundschaft resultierenden Rechte eingestellt werden können
oder
es ist möglich, angehörigen der selben Nation mehr Rechte zu verleihen, wobei Nationsfremden mit Zustimmung des NatFührers die selben Rechte zugestanden werden können.
Vorteile für das Spiel:
Freundschaften müssen nicht von Sicherheitsbedenken anhängig gemacht werden.
Nachteile für das Spiel:
Es wird komplizierter.
Mögliche Problematiken:
Es wird komplizierter.
