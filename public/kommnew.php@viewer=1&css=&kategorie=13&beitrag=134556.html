So ist es momentan:
Man kann gezielt eine Standardwaffe auswählen, jedoch nicht die bloße Faust.
So soll es werden:
Es erscheint die zusätzliche Option "ohne" im Dropdown-Menü, welche die Verwendung der mitgeführten Waffen außer Kraft setzt.
Vorteile für das Spiel:
Ein erheblicher Rückgang von Schnittverletzungen bei den Kneipenschlägereien auf Kathodos, infolge unbeabsichtigter Waffenbenutzung (da unsere Türsteher fast immer selber angetrunken sind, fallen die Einlasskontrollen eher lasch aus).
Nachteile für das Spiel:
Weniger zu kurierende Schnittverletzungen für die Heiler auf Kathodos.
Mögliche Problematiken:
Geschwollene, statt abgeschnittener Zwergennasen/Elfenohren...
