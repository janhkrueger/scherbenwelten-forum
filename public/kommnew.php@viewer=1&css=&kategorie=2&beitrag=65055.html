Angela Cada von Gondolin (RIP) *eine hübsche Halblingsdame kommt an da Anschlagsbrett, sieht einige Zeit die Anschläge durch und heftet dann selbst einen an:*

Werte Scherbenbewohner,

lange Zeit ist es nun her, dass mir meine Tochter Melis genommen wurde, wir hier auf der Scherbe getrennt wurden.
Sie war ein hübsches Kind, sie war so süß wie nur Honig sein kann.
Ihr rötliches Haar lockte sich sachte auf ihrem zarten Köpfchen. So klein sie doch war, so konnte sie einen doch anlächeln und einen erfreuen wenn sie mit ihren glau/blauen Augen einen ansah.

Ich weiß das sie noch lebt, mein kleines Kind.
Heute müsste sie etwa im Alter von 23 Jahren sein.

Sie hat auch ein paar besondere Merkmale, da wäre ein kleines Muttermal auf ihrer rechten Schulter, es ist süß klein und sah aus wie ein Herzchen.
Was auch auffallend ist, wenn man ihre Ohren ansah, das ihre Ohrläppchen angewachsen waren.
Trotzdem hatte sie die süßesten Ohren, die ein Baby nur haben kann.


So bitte ich euch, ihr Scherbenbewohner, helft mir meine Tochter zu finden, haltet ausschau nach einer jungen Dame, welche diese Merkmale tragen könnte.

gez.
Angela Cada von Gondolin
