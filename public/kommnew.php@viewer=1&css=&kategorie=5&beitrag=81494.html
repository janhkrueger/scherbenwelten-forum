Hymir (RIP) A. Internet Explorer

1. Klicken Sie auf den Button ´Extras´ in der Kopfleiste.
2. In dem sich öffnenden Fenster ganz unten auf ´Internetoptionen´ klicken.
3. Wenn sich das Fenster ´Internetoptionen´ geöffnet hat, klicken Sie im mittleren Abschnitt (´Temporäre Internetdateien´) auf ´Dateien löschen...´
4. In dem neuen Fenster ´Dateien löschen´ stellen Sie sicher, dass das Feld neben ´Alle Offlineinhalte löschen´ angehakt ist.
5. Nachdem Sie auf OK geklickt haben kann der Löschvorgangen einige Sekunden dauern.
6. Schliessen Sie alle Fenster. Das Löschen Ihres Caches ist beendet.
