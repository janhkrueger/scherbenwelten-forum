Zadar
So ist es momentan:
- max. 3 Energieschübe können täglich in AP gewandelt werden.
So soll es werden:
- max. 3 Energieschübe können täglich in AP gewandelt werden.
- Zusätzlich wird es möglich Energieschübe unbegrenzt als Zahlungsmittel für die AP Kosten in den 4 Basisschulen einzusetzen.
Vorteile für das Spiel:
- Energieschübe können in größerer Anzahl verwendet werden ohne das es zu größere Nebeneffekte gibt.
- Spieler mit hunderten Energieschüben wird ein Weg geboten diese innerhalb einer überschaubaren Zeit abzubauen.
Nachteile für das Spiel:
-
Mögliche Problematiken:
- Ausbildung wird erleichtert.
- Weniger Energieschübe verfallen.
