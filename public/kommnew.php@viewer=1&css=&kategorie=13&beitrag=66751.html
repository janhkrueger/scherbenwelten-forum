_Sonstiges - Traffic und CSS-Dateien
Lugenmaul (RIP) So ist es momentan:
Die CSS-Dateien werden auf allen Seiten eingebunden, d.h. es sind zu Spitzenzeiten die am häufigsten aufegrufenen Einzeldateien. Das führt öfters dazu, dass man nur auf eine einzige Datei wartet und der Seitenaufbau deshalb recht zäh wird.
So soll es werden:
CSS-Dateien, ähnlich den Bildern, zum Download und lokaler Speicherung anbieten und per Checkbox in den Optionen verankern.
Vorteile für das Spiel:
Indirekt. Vorteile für das Spiel: keine. Vorteile für die Benutzer: Hohe Serverlast macht sich erst in schweren Fällen bemerkbar. Der bisherige Lag-Standardfall, alles ist da, bis auf die CSS-Dateien, wird vermieden.
Nachteile für das Spiel:
vermutlich keine.
Mögliche Problematiken:
evtl. gibt es browserspezifische Problem beim Einbinden von lokalen Dateien, ähnlich der, per default deaktivierten, Einbindung lokaler Bilder in Mozilla. -> Deshalb sollte diese Funktion zunächst optional angeboten werden.
