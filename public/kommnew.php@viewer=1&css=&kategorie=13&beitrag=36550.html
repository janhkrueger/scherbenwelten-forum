Reformation Lehnssystem und Landnahme
Natas v. F. (RIP) Hier mein Vorschlag zur Reformation des Lehenssystems:

Man muss nicht mehr in einer Nation sein, um Felder zu requirieren.
Wenn man sie requiriert, werden sie dem eigenen Land zugeordnet, es geht in den eigenen Besitz über, welches dann Allod (Eigenland) genannt wird.
Wenn man in einer Nation ist, kann man es auch für die Nation requirieren, welches so behandelt wird wie bisher.
Für jedes zehnte Feld, was man als Allod hat, kriegt man pro Tag ein PP und abhängig von der Ertragsstärke ein Wenig Gold.
Auf diesem Allod kann man auch Steuern erheben.
Das nennt man Grundherrschaft.

Wenn jetzt jemand einen Lehnsmann nehmen will, kann er ihm etwas von dem Allod als Lehensland abgeben, welches so behandelt wird, als wäre es das Allod von dem Lehnsmann, aber wenn der Lehen gekündigt wird, geht das Lehnsland an den Lehnsherren zurück.
Bei den Nationen ist das Genauso:
Der Nationsführer/Nationsführerin kann Nationsland als Land an neue Mitglieder geben, welches, sobald diese austreten, wieder an die Nation zurückgeht.
Das nennt man Lehenswesen.
Ansonsten ist die die Bindung Lehnsmann - Lehnsherr wie die jetztige. Also ein Politikpunkt pro Lehnsmann.

Das würde das Spiel realistischer an das Lehns- und Grundherrschaftssystem des Mittelalters anpassen.
