Kleines Firefox Tuning für DSL nutzer
Jolina (RIP) Der sich weiterhin sich starker Zuwachsraten erfreuende Mozilla Firefox wird auch für seine Schlankheit und seine hohe Geschwindigkeit gelobt. Mit ein paar Tricks lässt sich aber noch einiges mehr an Speed aus dem Open Source Browser herausholen.

Vorteile

So können vor allem die BenutzerInnen von breitbandigen Internetzugängen erheblich von der Aktivierung des sogenannten "HTTP Pipelining" profitieren. Dabei werden die Anfragen an den Server nicht hintereinander sondern parallel verschickt, was die Netzwerkperformance dramatisch erhöht.

Aktivierung

Um HTTP pipelining zu aktivieren, müssen Sie in der Adresszeile des Browsers about :config eingeben, anschließend suchen Sie die Einträge "network.http.pipelining" und "network.http.proxy.pipelining" und setzen beide auf "true". Zusätzlich erhöhen sie den Wert von "network.http.pipelining.maxrequests" auf einen höheren Wert wie etwa 30, dieses ist die Zahl der maximalen gleichzeitigen Anfragen, die der Browser vornimmt. Anschließend starten Sie den Browser neu und erfreuen sich des Speed-Upgrades.
