System/Engine/Technisches - Verfall von Questgegenständen (1 )
Melisandra
So ist es momentan:
Z.b. ist das Bohnenpanzerquest beendet und man behält den Gegenstand in seinem Inventar.
So soll es werden:
Questgegenstände verfallen und informieren kurz vor dem Verfall per Diplo (wie bei Armeen z.b.), das dieses Quest bald beendet ist.

Z.b. könnte die Diplo so aussehen:
Auf deinem Wichteloptionsschein fängt auf ein mal ein kleines Datum zu leuchten, welches vorher nicht dort gewesen ist, es sieht wie ein Verfallsdatum aus und zeigt an, das irgendwas in 3 Tagen passieren wird.
Vorteile für das Spiel:
Man ärgert sich weniger, das man ein Quest nicht mehr machen kann.

Zum Ende eine Quest hin würden noch mal mehrere Leute dieses Quest machen => Das Quest was für die Spieler gemacht wurde, erzeugt noch mal etwas von dem Spass wozu es gedacht war
