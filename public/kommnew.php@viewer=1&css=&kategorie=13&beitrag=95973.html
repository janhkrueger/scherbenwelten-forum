So ist es momentan:
Fischerhütte gibt es nicht.
So soll es werden:
Jetzt kommts:
Das Gebäude "Fischerhütte" soll erschaffen werden.

Fischerhütten können in einer maximalen Entfernung von 2 Feldern von einem Anleger gebaut werden.

Baukosten entsprechen denen einer Jagd/Holzfällerhütte.
Als Besatzung werden Seeleute gebraucht( oder andere mit entsprechenden Fischen-Skill)

Zur Steigerung der Produktion können Reusen ausgebaut werden.

Nach dem Kauf eines Rezeptes kann in der Fischerhütte Tran gekocht werden.
Vorschlag: 100 Fisch+ 5 Holz = 2 Tran pro Prodrunde

Dieses Rezept kommt aber erst zum Einsatz wenn der Arbeiter in der Hütte einen Mindestskill im Fischen von 50 hat.
Vorteile für das Spiel:
1.Der Tranmarkt würde sich ein wenig entspannen ohne aber gleich zusammen zu brechen da wir ja keine Massenprod erreichen.

2.Der Seemann als Beruf würde ein wenig aufgerwertet.

3.Fisch als Ware würde mal eine kleine Aufwertung erhalten da er mal zu was nütze wäre ausser an Halblinge verfüttert zu werden.

4. Das "Tranmonopol" wird ein kleines bischen aufgeweicht.
5. Eventuell werden Sonderwaren für die Tran gebraucht wird, mal etwas erschwinglicher.
Nachteile für das Spiel:
Ausser das einige "Festungsbauer" rumheulen werden das sie wegen ihrer milionenschweren Bollwerke um die Anleger keine Fischerhütten bauen können, sehe ich auf Anhieb keine Nachteile
