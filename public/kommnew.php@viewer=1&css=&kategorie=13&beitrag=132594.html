Fabius Alagos
So ist es momentan:
Feindliche Konflikte werden zumeist über einseitig operierende Armeen oder EinzelPvPs ausgetragen.
So soll es werden:
Es gibt eine Option für Massenpvps. Für 150 PPs kann ein hoher Titelträger/Nationsführer/Verkünder ein Schlachtfeld auf einem beliebigen Feld anlegen (hier sind ab diesem Zeitpunkt keine Einzelwds/rüs/hhte und kein Abstellen von Armeen mehr möglich. Die Herausforderung kann durch einen Feind ebenfalls durch PPs angenommen werden. Beide Seiten haben dann die Möglichkeit x Leute ins Feld zu führen, indem sich die Spieler ähnlich wie bei einem WD "aufs Schlachtfeld begeben" für 1k APs und auf die Auslösung warten.

Es kann um eine einstellbare Sache gekämpft werden z. B. Verfall der NatKE, Geldübergabe oder auch RP-Dinge die der Verlierer dann selbst einstellen müsste: z. B. Eintritt in die Lehenskette des anderen, Kapitulation, Übergabe von Städten, Land etc.

Die Auswertung erfolgt an einem einstellbaren Datum/Uhrzeit/Rundenauswertung/was-weiss-ich.

Es wird solange gekämpft bis auf einer Seite keine Überlebenden mehr da sind (die Toten landen normal im Tempel). Ein Remis wäre auch denkbar.
Vorteile für das Spiel:
Schönes RP möglich. Kriege würden nicht mehr soviel Geld kosten :)
Nachteile für das Spiel:
Beide Seiten müssten irgendwo miteinander spielen, so dass berechtigte Zweifel bestehen, ob die Sache genutzt würde.
Mögliche Problematiken:
Der Server knickt ein.
Sonstiges:
Toll wäre es auch wenn die Schlacht nicht in wenigen Sekunden vorbei wäre, sondern man die einzelnen Schlagabtäusche wegen mir im 3-5 Minutentakt verfolgen könnte. = SW-Kino
