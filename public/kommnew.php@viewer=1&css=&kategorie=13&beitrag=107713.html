Gebäude/Rezepte/Produktion - Landsteuern verweigern
Dracon Darknight (RIP) So ist es momentan:
Landsteuern können eingehoben werden, gleich ob die Felder von Nationen oder Monstern besetzt werden
So soll es werden:
Es sollte eine Einstellung geben die es dem Spieler erlaubt seine Steuerabgaben zu verweigern.

In diesem Fall besteht eine x%tige Chance, dass die Gebäudeinsassen von der Monsterasse der das Feld gehört oder einer Gruppe Steuerintreiber der Besitzernation angegriffen wird.

Für die Kampfberechnung sollten Wachen und Waffen des Gebäudes halbiert werden.
Wird der Kampf verloren nehmen die Steuereintreiber allerdings den dreifachen Warenwert mit.
Vorteile für das Spiel:
-) Ein weiteres, nettes RP-Element
-) Spiellogik
-) Vorteile für kleinere Nationen, die nicht ständig mit
Monstern um die Gebiete kämpfen können
Nachteile für das Spiel:
Eventuelle Steigerung der Inflation, da weniger Waren aus Gebäuden abließen könnten
