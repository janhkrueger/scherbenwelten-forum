Tharsonius (RIP) So ist es momentan:
Jemand fängt in SW mit einem Schnupper-Acc an. Er startet mit einem Char, 0 Ruhm, 0 Geld, nichts. Nach 4 Wochen soll er dann ein Abo abeschließen um sich in die große weite Welt aufzumachen.
So soll es werden:
Schließt jemand mit einem Schnupper-Acc ein Abo ab bekommt er Ruhm geschenkt.
Monatsabo 10% des Durchschnittsruhms im Spiel
Vierteljahrabo 20%
Halbjahresabo 30%
Jahresabo 50%

Durchschnittsruhm wie der Name schon sagt. Der durchschnittliche Ruhm aller aktuellen Spieler in SW. Keine Ahnung wie hoch der Durchschnittswert gerade ist.

Für Hardcorefans das ganze vielleicht nur als eine Option die man auch ablehnen kann. So kann jeder entscheiden ob er den Ruhmbonus nimmt oder lieber von 0 anfangen will zu spielen.
Vorteile für das Spiel:
Ein Anreiz für einen Neueinsteiger ein Abo abzuschließen. Er muss dann nicht komplett von 0 alles aufholen. Dadurch dann vielleicht doch mal mehr Spieler die nicht davon abgeschreckt werden alles von 0 an zu erarbeiten. Der Durchschnitt müsste irgendwo bei 7-8 Chars in der Laufgruppe/Prodgruppe liegen beim Jahresabo.
Nachteile für das Spiel:
Es gibt keine Leute mehr die mit einem Char starten? Die Idee ist unausgegoren, ich weiß. Nachteile fallen mir keine ein. Nrr Benachteiligungen gegenüber den Leuten die ihre Abos vorher abgeschlossen haben und von 0 an auf spielen mussten.
Mögliche Problematiken:
Programieraufwand, was weiß ich noch alles.
