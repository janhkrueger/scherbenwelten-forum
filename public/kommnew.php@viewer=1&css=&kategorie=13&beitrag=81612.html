Gebäude/Rezepte/Produktion - Handelsposten in Innenstadt
Turiandor Dragus
So ist es momentan:
Handelsposten können nur außerhalb von Städten errichtet werden.
So soll es werden:
Handelsposten können auch in der Innenstadd errichtet werden, ähnlich wie die Universitäten, Brauhäuser, Schmieden und Bäckereien.
Vorteile für das Spiel:
Man könnte die Innenstadt durch ein kleines Handelsviertel beleben.
Nachteile für das Spiel:
Im Spiel könnten bestimmte Parteien durch ein Stadttor von den Handelsposten ausgeschlossen werden, jedoch ist dies ja auch duch eine KE außerhalb der Stadt der Fall.
Mögliche Problematiken:
Etwas Aufwand es umzustellen.
Sonstiges:
Es könnte auch ein neues Gebäude eingeführt werden, wie z.B. der "Markt", welcher die Selben Funktionen hat wie ein Handelsposten, aber eben eher an eine Innenstadt angepasst ist.
