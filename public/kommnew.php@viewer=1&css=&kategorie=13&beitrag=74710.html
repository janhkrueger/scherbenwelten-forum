Glauben/Konvente - Begrenzung Machteinsatz (4 )
Bortas (RIP) So ist es momentan:
In einer Zeit, in der die Konvente über teilweise 7-stellige Machtbeträge verfügen, wird nicht kleinlich mit dem Machteinsatz umgegangen. Teilweise werden Leute auf einen Schlag direkt für den Rest ihres Scherbenlebens verketzert, ohne Hoffnung, je wieder den Ketzerstatus loszuwerden. Der einzige Grund dafür: Weil Macht im Überfluß da ist, und man ruhig großzügig damit umgehen kann.
So soll es werden:
Festlegung einer Höchstgrenze von maximal 10.000 Macht pro Antrag, besser wären vielleicht sogar 5000. Wenn doch mal jemand länger Ketzer sein soll, muß halt wieder ein neuer Antrag gestellt werden.
Vorteile für das Spiel:
1. Spieler können nicht mehr so leicht "lebenslänglich" verketzert werden.
2. Einem Konvent kann nicht mehr durch einen einzigen sinnlosen Antrag die gesamte Macht geraubt werden.

keine

keine
