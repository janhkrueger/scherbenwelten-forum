Tharsonius (RIP) So ist es momentan:
Eine Armee kann nicht durch einen Tunnel mitgenommen werden.
So soll es werden:
Tunnel können mehrfach ausgebaut werden indem ein weiterer Tunnelanbau auf ein schon existierendes Tunnelstück rübergebaut wird. Diese weiteren Ausbauten können von jedem gemacht werden der in den Tunnel reinkommt, also auch Diebe die in so ein Gebäude einbrechen.

Durch einen normalen Tunnel können wie bisher nur Chars. Durch einen einfach ausgebauten Tunnel können Infantriearmeen, durch einen zweifach ausgebauten Tunnel Kavalleriearmeen, durch einen dreifach ausgebauten Tunnel können Katapultarmeen. Sprich um eine Kataarmee durch einen Tunnel zu schicken muss jedes Feld 3 mal überbaut worden sein.

Der Gebäudebesitzer soll selbstverständlich eine Meldung bekommen das in seinem Tunnel bauarbeiten stattfinden. Am Günstigsten wäre aber wenn dabei kein Name auftaucht.
Vorteile für das Spiel:
Diese elenden "Tunnelfestungen" können auch mal geknackt werden. Das Antunneln von Gebieten um dort uneinnehmbare Basen aufzubauen ist in SW inzwischen ja Volkssport geworden. Jeder gräbt sich ein und hockt dann dumm rum und kläfft wie ein Köter aus seiner sicheren Hütte andere Leute voll. Ein ewiger Grabenkrieg der die Scherbe langsam aber sicher immer weiter erstickt.

Eine neue Möglichkeit Waren von der Scherbe abfließen zu lassen.
Nachteile für das Spiel:
Es könnten wieder mehr Sachen von Spielern zerstört werden und damit die Flamequote im Forum noch weiter ansteigen. Ach ich sehe ehrlich gesagt keine Nachteile außer das Leute anfangen rumzumaulen weil ihre Festungen nun doch noch angreifbar werden.
