So ist es momentan:
Man kann nur eine bestimmte Menge Armee AP eintauschen gegen PP pro Tag, das führt dazu das Kriege fast immer zur Hauptrunde geführt werden, und damit die Regelung eigentlich umgangen wird. Und 2. man bei grösseren Angriffen darauf vertrauen kann das die Angriffe zu dieser Zeit erfolgen.
So soll es werden:
Es soll ein Timer eingebaut werden das zwischen zwei mal eintauschen der maximalen Armee AP 24 h liegen müssen. ( oder auch 12h oder 18h)
Vorteile für das Spiel:
größere Kriegsaktionen verteilen sich über den ganzen Tag
Nachteile für das Spiel:
Angriffe werden schwerer da für einen Angriff weniger Armee AP zur verfügung stehen ( könnte man möglicherweise etwas ausgleichen in dem die Anzahl der eintauschbaren AP leich erhöht wird)
Mögliche Problematiken:
sehe keine
Sonstiges:
Würde damit allen die Gelegenheit geben sich an solchen Aktionen zu beteiligen, da diese nicht immer auf die Hauptrunde fallen würden.
