Grafiken - Ein Platz für Bürgermeister (42 )
Glorfindel von Gondolin (RIP) So ist es momentan:
Durch das neue Grafikpaket ist das Viereck als Grafik weggefallen. Das ist zwar im Prinzip toll, weil die Mauerkreuzung sich schon viel besser in die Stadtplanung einbinden lässt.

Auf der anderen Seite wurde das Viereck auch gerne als Bastei oder Torturm eingesetzt wo die Kreuzung doch eher unmöglich aussieht. Außerdem war das Viereck auch traditionell das Söckelchen für den Bürgermeister, so dass jetzt hunderte von Bürgermeistern ohne Platz zum schlafen haben.
So soll es werden:
Es soll eine zusätzliche Grafik geben, die dann die Funktionen des alten Vierecks übernehmen kann und die zu den neuen Innenstadtgrafiken passt.

Ob das dann wieder ein reines Viereck ist, oder tatsächlich nette hübsche (und funktionslose) Tortürme überlasse ich den Künstlern unter uns. Ich kann sowas leider nicht.
Vorteile für das Spiel:
Die Innenstädte werden hübscher und Bürgermeister haben wieder einen angemessenen Platz zum schlafen.
Nachteile für das Spiel:
Mir fällt keiner ein.
Mögliche Problematiken:
Die Grafik müsste so gestaltet werden, dass sie von allen Seiten an Mauern angeschlossen werden, aber auch alleine stehen kann.
