Leta (RIP) Nínael Fael:
>> < font color=4876FF > Farbe (Bsp. Hellblau)

Font ist in zukünftigen HTML Standards nicht mehr enthalten und sollte nicht mehr verwendet werden.

<span style="color:#BBBBFF"></span>

Der Text dazwischen wäre z.B. ein recht helles Blau. Und Parameter kommen immer zwische ".

<big> und <center> werden von den Browsern unterschiedlich interpretiert. Big wäre font-weight:bolder; in CSS, zumindest nach der IE Interpretation. Aber im Mozilla ist es font-weight:boldest;. Center zentriert im IE alle darin enthaltenen Dinge. Im Opera nur Text.
