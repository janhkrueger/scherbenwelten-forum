Abel von Pavenacht (RIP) Bei Standartrezepten ( also Strassenkleidung bis Plattenpanzer) wirkt sich das Kampfverhalten auf den Rüstungsschutz aus.

Das bedeutet, man bekommt beim "vorsichtigen" Kampf einen Bonus von 2 Rüstungsstufen. Das bedeutet eine Strassenkleidung wirkt wie eine Stufe 3 Rüstung, eine Fellkleidung wie eine Stufe 4 Rüstung.

Bei "normalem" Kampfverhalten zieht der orignal Wert der Rüstung ( Strassenkleidung Stufe 1, Fellkleidung Stufe 2 usw).

Bei "rücksichtslosen" Kämpfen werden 2 Rüstungsstufen abgezogen. Das bedeutet, dass Rüstungen wie Strassenkleidung und Fellkleidung ignoriert werden, da ihre Stufe dadurch auf bzw unter 0 fällt. 		29.01.04 16:34
