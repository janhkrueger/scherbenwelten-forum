Morgoth Turambar (RIP) So ist es momentan:
Wenn an irgendeinen Mitspieler Geld gezahlt wird, dann bekommt der Mitspieler eine Meldung, das er Geld bekommen hat.
Man selber bekommt aber keine Meldung das man eine bestimmte Summe gezahlt hat.
Also Spieler X zahlt an Spieler Y eine Summe Z.
Spieler Y bekommt die Meldung:Spieler X zahlt Z Goldstücke an Dich.
Spieler X bekommt gar nichts.
So soll es werden:
Beide Spieler sollten eine Meldung bekommen, wer an wen wieviel Geldstücke bezahlt hat.
Vorteile für das Spiel:
Kontrolle, ob man dem anderen Spieler die richtige summe gegeben hat.
Bei größeren Zahlungen mit vielen Nullen kann man sich schon mal vertun.
Sicherheit bei Geschäften: Wenn es Streit gibt wegen Bezahlungen kann man nachweisen, das man die Summe Z bezahlt hat.
Nachteile für das Spiel:
Mehr Systemmeldungen
