Johannes Gualbertus (RIP) So ist es momentan:
Wenn man eine Stufe in einer der 3Gilden aufgestiegen ist, ist das nciht mehr rückgängig zu machen. Ein SW Neuling probiert alle 3 Stufen je eine aus und ist so daran gehindert für 32k GS in eine der drei Gilden zum Abschluß zu kommen. Die zweite Gilde ist dann auch fast unerreichbar.
So soll es werden:
Man sollte aus einer ungewollten Gilde wieder aussteigen können, um in einer anderen Gilde bis ganz nach oben zu kommen. Um das nicht mehrfach ausnutzen zu können, sollte man jede Gilde nur ein Mal verlassen können.
Vorteile für das Spiel:
Vieler der Anfänger könnten so zum Anfang eine Burg bauen oder 3Handelsrouten installieren und das für nur 32k GS.
Nachteile für das Spiel:
Durch die Begrenzung auf ein Mal Rückzug pro Gilde sehe ich keine Nachteile
