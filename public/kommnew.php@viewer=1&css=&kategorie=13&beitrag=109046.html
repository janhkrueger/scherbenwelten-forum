Gwydion de Ville (RIP) So ist es momentan:
Man kann mit Berufslosen XP sammeln bis zum abwinken und ihnen so für später Fertigkeiten verpassen, die eigentlich Anti-Skills sind.
So soll es werden:
Berufslose können jede Fertigkeit maximal bis 20 Basis ausbilden. Dadurch wird ein übertriebener Nutzen reduziert und die nicht vorhandene Logik, dass ein Berufsloser besser ist als ein Meister seines Faches in einem Gebiet wird beseitigt.
Vorteile für das Spiel:
- Die Klassen haben wieder eher Sinn
- Powergaming wird erschwert
- das Spiel wird wahrscheinlich wieder in die erdachten Bahnen gelenkt
Nachteile für das Spiel:
- Berufslose benötigen rasch eine Ausbildung
Mögliche Problematiken:
Aufschrei mancher Spieler nach diesem Vorschlag;
Gleiches Recht für alle?
