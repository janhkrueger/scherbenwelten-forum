System/Engine/Technisches - Neues Fenster statt neuer Tab
Iga Pelogran (RIP) So ist es momentan:
Klickt man auf einen Namen in der History oder FE-Liste öffnet sich ein neues Diplofenster.
Gibt man den Namen jedoch per Hand ein und klickt auf den Button öffnet sich im FF und IE nur ein neuer Tab. Dabei wird das Fenster auf die Diplo-Größe resized. Schliess man dan Tab oder Sendet die Nachricht ist auch das Hauptfenster in dem SW läuft so klein wie das Diplofenster und muss erst wieder groß gemacht werden.
So soll es werden:
Der Verfassen-Button öffnet ein neues Fenster, keinen Tab/Registerreiter
Vorteile für das Spiel:
Das nervige Browsergröße per Hand anpassen fällt weg. Ich spiele halt nicht im "Vollbild" sondern habe mein Browserfenster an eine bestimmte Größe angepasst. Gerade wenn man öfters an nicht Freunde schreibt, die gerade offline sind (also wo man keinen Link zum Anklicken hat) ist das nervig.
