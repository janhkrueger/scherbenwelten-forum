Mibo van Draaken (RIP) http://kartenarchiv.de.vu/
Pro volle 1000 Einwohner kann ein Gebäude um eine Stufe ausgebaut werden. Ein Ausbau verfällt, wenn das Ausbautenmaximum (in Folge von Stadtschrumpfung) kleiner als 3/4 wird oder weniger als 3/4 der zum Unterhalt benötigten Arbeiter zur Verfügung stehen. Der 20. Ausbau beispielsweise verfällt wenn die Einwohnerzahl der Stadt unter 15k fällt.

Damit sollten von deinen 110 Ausbauten erst was verfallen wenn du unter 82500 Einwohner kommst ;-) 		25.01.06 11:39
