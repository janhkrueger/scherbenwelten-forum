Errichten
Möchtet Ihr ein Gebäude errichten, müsst Ihr zunächst den passenden Untergrund suchen. Eine Holzfällerhütte kann beispielsweise nur in einem Wald errichtet werden, ein Steinbruch nur im Gebirge, usw.
Habt Ihr eine geeignete Stelle gefunden, dann klickt auf "Gebäude errichten". Es erscheint dann eine Liste der Gebäude, die an diesem Standort errichtet werden können. Klickt nun auf das Gebäude, das Ihr bauen wollt.
Es erscheint nun eine kurze Beschreibung des Gebäudes und die Baukosten. Wenn Ihr dann auf "Bau starten" klickt, wird das Gebäude im nächsten Augenblick erstellt, sofern Ihr die notwendigen Materialien zur Verfügung habt.

Zusätzlich zu den Rohstoffkosten erhebt der König Lehenskosten, je nachdem wieviele Gebäude ihr bereits besitzt. Diese werden vor dem Bau angezeigt und steigen je Gebäude um das 10-fache an.

Die Errichtung von Gebäuden kostet zusätzlich zu den Rohstoffen 1000 AP. 		29.03.06 19:31
