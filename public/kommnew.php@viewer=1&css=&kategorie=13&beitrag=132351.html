Gebäude/Rezepte/Produktion - Automatische Wahl des Transferz
Kapo el Ligno
So ist es momentan:
Wenn man im "Warenhandel" im HP oder auf dem Stadtmarktplatz ist, ist immer die Gruppe als Transferziel voreingestellt
So soll es werden:
Wenn auf einer Seite ein Wert verändert wird (also eine Warenmenge eingetragen wird), wird die andere Seite als Transferziel eingestellt (kann aber weiterhin per Maus oder Tastatur verändert werden).
Vorteile für das Spiel:
Man kann eine Warenmenge eintragen und einfach "Enter" drücken.
Nachteile für das Spiel:
Ich sehe keine
Mögliche Problematiken:
Wer gewohnt ist, nach dem Eintragen der Werte auf der linken Seite zum Auswahlfeld für das Transferziel zu fahren, wird eine Überraschung erleben.
Sonstiges:
Keine Serverbelastung, die Änderung ist ein triviales Stück Javasgript. 		12.07.12 0:49
