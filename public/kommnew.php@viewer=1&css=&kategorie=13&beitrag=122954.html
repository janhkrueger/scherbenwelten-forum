So ist es momentan:
Angriffe auf gut geschützte Ziele können praktisch nur anhand von Armeen erfolgen.
So soll es werden:
Angriffe von PvP-Gruppen machen ebenfalls wie Armeetruppen einen Trampelschaden, ebenso die verteidigenden Gebäudeinsassen. Möglicherweise müsste man die Trampelstärken etwas dahingehend regulieren, dass z. B. ab 15 Chars Trampel tödlich wirken können.
Vorteile für das Spiel:
SW ist so ausgelegt, dass man zusammenarbeiten muss um etwas zu erreichen. So sollte es auch bei Kampfhandlungen sein. Es gäbe keine so sicheren Gebäude mehr wie derzeit.
Nachteile für das Spiel:
Spieler die sich "sichere" Plätze geschnappt haben erfahren einen Nachteil.

Mögliche Problematiken:
Angriffe zu Nachtzeiten. Unterstützung durch Sittaccounts sollte nicht möglich sein (funktioniert meines Wissens aber auch jetzt schon nicht).
Sonstiges:
Sicherlich ein gewagter Vorschlag von jemandem der von der Änderung profitieren würde und der glaubt, dass kriegerische Auseinandersetzungen im Spiel für Spielspaß sorgen und es davon in den letzten Jahren sehr wenig gegeben hat (da kann man durchaus auch anderer Meinung sein).

Ich glaube nicht, dass ein wildes Hauen und Stechen losgehen würde, wohl jedoch dass es eine Belebung geben würde.
