DeLaCroix (RIP) So ist es momentan:
Irgendwann geben Monster 0 XPs, zumindest, wenn man sie schon sehr sehr häufig besiegt hat.

Sammeln gibt immer 5/7/9 XPs? Aber definitiv eine konstante Menge.
So soll es werden:
Entweder gibt sammeln irgendwann auch 0 XPs, das wäre fair,

oder, was mir lieber wäre,

Monster geben immer min. 1 XP.

Vorteile für das Spiel:
Wenn ich XPs brauche, muss ich nicht zwischendurch 2 Wochen sammeln gehen, sondern kann mich den ganzen Fledermäusen und Riesenratten widmen, die so rumschwirren.
Nachteile für das Spiel:
Eigentlich keine, denn wo die XPs herkommen ist wurscht, ob nen paar Fledis weniger oder 3 abgeerntete Felder um eine Stadt herum, da beisst die Maus keen Faden ab.
