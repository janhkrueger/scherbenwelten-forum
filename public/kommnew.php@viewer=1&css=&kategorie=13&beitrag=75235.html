So ist es momentan:
Alle Gebäude müssen Schnee für ihre Lagerausbauten über 50 ausgeben dies führt z.T. dazu das einfach mehrere Produktionsgebäude des gleichen Typs von einem Spieler errichtet werden um hohe Lagerkosten zu umgehen.
So soll es werden:
Es wird in Massenwarenproduktionsgebäuden eine zweite Lagerart eingeführt die nur die jeweilige Massenware lagern kann aber keinen Schnee kostet.
Vorteile für das Spiel:
Attraktivere Massenwarengebäude deren Besitzer noch nie lange Reisen konten.
Wieder weniger Massenwarengebäude von einem Spieler.
Nachteile für das Spiel:
Geringerer Schneebarf.
Mögliche Problematiken:
Bereitsvorhandenes Lager könnte in die neue Lagerart auf Wunsch umgewandelt werden.
