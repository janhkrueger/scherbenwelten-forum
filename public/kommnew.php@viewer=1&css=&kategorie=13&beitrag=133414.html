So ist es momentan:
Scherbenwelten ist in die Jahre gekommen und es ist schade, dass viel Wissen einfach verloren geht.
Bei den einzelnen Chars werden nur sehr wenige Ereignisse aufgezeichnet:
62.Saatmond 407, Thrilmambor Lómumir wurde geboren!
6.Dunkelfrost 427, Thrilmambor Lómumir begleitet nun Ajnin
8.Dunkelfrost 427, Thrilmambor Lómumir wird Jäger
Ereignisse der Gruppe werden nicht aufgezeichnet.
So soll es werden:
Charakterhistorien wie z.B. Nationseintritte, -austritte (ggf. auflösungen), Konventszugehörigkeiten, Ränge Kriegergilde/Handelsgilde/Diplogilde werden festgehalten. Diese sind einsehbar ähnlich zu Statistiken.

Eventuell können anderen diese Historie ausspionieren.
Vorteile für das Spiel:
Wow-Effekt für Langzeitspieler (war ich wirklich mal in einer Nation mit so einem doofem Namen ;))
Belebung des RP, wenn man auf diese Daten zurück greifen kann
Nachteile für das Spiel:
Keine gefunden
Mögliche Problematiken:
Wenn Spieler Einträge aus enginetechnischen Gründen erzeugen (Nateintritt, Felder reggen, Nataustritt) wird diese Historie übersichtlich und lang.
