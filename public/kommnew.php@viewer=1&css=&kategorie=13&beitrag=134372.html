Kampf/Armeen/Monster - Reggeschwindigkeit von Monstern (21 )
Lisa von Falkenstein (RIP) So ist es momentan:
Monster reggen im Vorbeilaufen. Hat eine Spinne erstmal Witterung aufgenommen und rennt einem mal durchs Land ist das ganze Land innerhalb weniger Stunden von Spinnen eingenommen.
So soll es werden:
Auch Monster müssen wie Spieler reggen. Sprich die Berechnung der Dauer für das Reggen der Felder durch Monster wird Berechnet in dem man Benötigte AP (wenn es ein Spieler erobern wollte (gerade bei Städten interessant) geteilt durch 5. Soviele Minuten dauert es bis das Monster das Feld gereggt hat. Kommt also ein Monster auf ein Feld dass es für 1000 AP reggen könnte (wenn es ein Spieler wäre) dann würde es 1000 / 5 = 200 Minuten dauern, also 3 Stunden und 20 Minuten. Ein Spieler der 300 AP/Stunde regeneriert bräuchte genau so lange.
Vorteile für das Spiel:
Das Reggen von Feldern steht wieder in einem "guten" Verhältnis zum verlieren der Felder durch Monster.
Nachteile für das Spiel:
-
Mögliche Problematiken:
Wie soll entschieden werden ob ein Monster weiterreist (um schnell vorranzukomen) oder reggt und sich dann ne Ewigkeit nicht bewegt...
