Er kam gerade von einer Reise zurück zu seiner Holzhütte als er feststellte, dass eine Gruppe Orks seine Hütte sehr gründlich durchsuchte. Mutig stellte er die Gruppe, doch anstatt anzugreifen traten die Orks geschickt den Rückzug an.

So gründlich wie sie das Haus durchsucht haben, muss es sich um etwas sehr kleines gehandelt haben.
