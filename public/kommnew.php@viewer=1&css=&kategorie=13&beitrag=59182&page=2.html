Noch ein paar Überlegungen (nicht ganz ausgereift), wie man das etwas ausbauen kann:

Option: Für jede Person, die der Bürgermeister auf die Ruhmliste setzen möchte zahlt er die Anzahl der bestehenden Personen einmalig bei Nennung in Politikpunkte. Erste kostenlos, Zweite 1PP  wie beim Gladiator. Oder eben nicht PPs aber der AP-Verbrauch zum anlegen steigt.

Option: Es wird nicht die Menge an Ruhm eingestellt, sondern nur der Name angegeben und diese Person bekommt fix 100 Ruhm. Jede weitere Person muss mit PP hinzugefügt werden. Einer kann eventuell auch mehrfach angeführt sein, kostet aber PPs. Wäre auch

Option: Ein Historieeintrag, damit es die Wichtel mitbekommen. Damit sehen alle, wer so Ruhm bekommt (X wurde von der Stadt geehrt, x hat von der Stadt eine Auszeichnung erhalten, )

Option: Wenn Ruhm so verteilt wird, dann wird die verteilte Menge um 10% reduziert. Beispiel: Um einer Person 100 Ruhm zu geben muss der BM auf 111 Ruhm verzichten.

Option: Jede Ruhmverteilung aus der Stadt ist mit einem Titel verbunden. Damit wird in der Historie auch der Grund für diesen Ruhm ersichtlich. (X wurde zum königlichen Hoflieferanten für Bier und Wein ernannt. X wurde zum gräflichen Heerführer des Landschutzregiments ernannt. ).
