@Tillion ...Völliger Unfug (hast du überhaupt eine Pumpe?):

Alchemie und Steinklopfen zählt zusammen. Entscheident für die Produktion ist der Gruppeskill Alchemie+Steinklopfen.

...den tust du unten in die Formel rein und schon hast du die Produktion.

floor( 2 + (steine skill + alchemie skill) * (10+ausbauten) / 10 * 0.2 )

Louison Dírnaith (RIP) Wenn man Schürfer reinstellt, hat man sehr schnelle eine gute Produktion, wenn aber der Schürfer seine 100 Basispunkte hat, dauert es sehr lange, bis er auf die 100 Basispunkte bei Alchemie kommt, denn dann ist er ein 100% Pumpenarbeiter.

Ein Alchemist (Magier/Heiler) hat bei Alchemie schon Basis 25 und man kann ihn manuell recht schnell im Steineklopfen trainieren.

Was man ausserdem beachten sollte :
der Malus bei Zwergen/Halblingen bei der Seefahrt, wenn man selbst fährt und die Arbeiter zur Schulung von der Pumpe in eine Stadt bringen muss.

der Bonus bei Halblingen beim Sammeln, ohne das man wohl kaum drum herum kommt, wenn man seinen Arbeiter Erfahrung beibringen will.

Beispiel Ausbildung
Zwergenschürfer -> Schürfen 25-100 : 94.500 XP -> Alchemie 0-100 : 252.500 XP = 347.000 XP

Halblings-/Menschenschürfer -> Schürfen 25-100 : 118.125 XP -> Alchemie 0-100 : 252.500 XP = 370.625 XP

Elfenschürfer -> Schürfen 25-100 : 141.750 XP -> Alchemie 0-100 : 252.500 XP = 394.250 XP

Zwergenalchemist -> Alchemie 25-100 : 118.125 -> Schürfen 0-100 : 227.250 XP = 345.375 XP

Halbling-/Menschenalchemist -> Alchemie 25-100 : 118.125 -> Schürfen 0-100 : 252.500 XP = 370.625 XP

Elfenalchimist -> Alchemie 25-100 : 118.125 -> Schürfen 0-100 : 277.750 XP = 395.875 XP
