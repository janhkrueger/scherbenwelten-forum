Celebrimbor (RIP) So ist es momentan:
Wenn man Diplos löschen möchte, geht das auf mehreren Wegen. Man kann:

- sie einzeln löschen
- sie alle löschen
- zu löschende Nachrichten markieren und die markierten löschen
- verschiedene Arten von Nachrichten löschen

Unter Notizen geht das jedoch nicht. Jede Notiz muss einzeln gelöscht werden. Dadurch das sich hierbei um einen normalen Link mit Übergabevariable handelt wird die Seite neu aufgebaut und man muss immer wieder nach unten scrollen. Falls man jetzt viele Notizen angesammelt hat, ist das Löschen sehr zeitintensiv.
So soll es werden:
Man kann wie bei Nachrichten alle löschen oder mit einer Checkbox mehrere markieren, die man löschen möchte.
Vorteile für das Spiel:
- weniger Klickarbeit
- Spieler kann einfacher „aufräumen“
Nachteile für das Spiel:
-/-
