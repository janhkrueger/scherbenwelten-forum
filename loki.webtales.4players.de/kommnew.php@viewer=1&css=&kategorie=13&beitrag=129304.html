Agesha Goldsucher (RIP) So ist es momentan:
In der Nationsbekanntmachung kann man zwar viel mit HTML gestalten, ein anklickbarer Link jedoch (wie z.B. zu einem externen Forum) ist leider nicht möglich.

Das spielinterne Forum bietet nicht so umbedingt die Möglichkeit für zb grosse Planungen (ausser man baut das Forum riesig aus). Also hat man ein externes Forum dafür eingerichtet damit IG nur IG-Dinge stehen.

So soll es werden:
Man kann in der Nationsbekanntmachung anklickbare Links einfügen. Gerne auch beschränkt auf 1-2 Links damits nich überhand nimmt.
Vorteile für das Spiel:
Wenn man nun in die Nationsbekanntmachung einen anklickbaren Link einfügen kann ist das eine gute Möglichkeit diejenigen gleich in den Beitrag der aktuellen Diskussion z.B. zu leiten. Man muss nicht erst den Link kopieren, Forum öffnen, anmelden, Link einfügen (oder den Beitrag so von Hand im Forum suchen).

Natürlich soll man nicht nur auf externe Foren verweisen können. Halt auf das worauf man grad hinweisen will.

Wie gesagt eine beschränkung der zu setzenden Verlinkungen wäre nicht verkehrt.
