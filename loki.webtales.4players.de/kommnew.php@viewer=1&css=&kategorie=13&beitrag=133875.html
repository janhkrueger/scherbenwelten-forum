So ist es momentan:
Zum Tausch von Städten muss ein Zwischenhalter eingeschaltet werden, was in Zeiten, wo nahezu jeder eine Stadt hat, unnötig verkompliziert.
So soll es werden:
Unter das Feld "Neuer Bürgermeister:
(Vertretung):" kommt ein Feld "Städtetausch", per Drop-Down kann der Name ausgewählt werden, stimmen die Namen gegenseitig, findet ein Tausch der Städte statt.
Vorteile für das Spiel:
Anpassung an die Spielrealität: Zu wenige Spieler ohne Städte.
Nachteile für das Spiel:
Weniger Beschäftigungsmöglichkeiten, sich irgendwo eine vertrauenswürdige Person ohne Stadt aus den Fingern zu saugen.
Mögliche Problematiken:
