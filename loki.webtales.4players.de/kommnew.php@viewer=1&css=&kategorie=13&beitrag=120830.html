Iga Pelogran (RIP) So ist es momentan:
Wird das Gebäude abgerissen, von dem aus ein tunnel gebaut wird, ist der Ganze Tunnel weg... Schmerzhaft zu sehen im aktuellen Zwergentunnel als Beispiel.
So soll es werden:
Wird das Gebäude zerstört, verfalln alle Anbauten im Radius von 2 Feldern (die zu dem Tunnel gehören), ergo 8 Anbauten.
Wird das Gebäude wieder Aufgebaut so kann man mit 8 Anbauten den Anfang wieder herstellen.
Vorteile für das Spiel:
Logik. Nur weil ich einen Tunneleingang zerstöre, schüttet sich doch nicht der Ganze Schacht automatisch zu...

Ich sehe keine.
