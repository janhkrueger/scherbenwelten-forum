Das Problem liegt, wie vermutet, im verwendeten StyleSheet. In diesem wird für Input-Felder im allgemeinen die Hintergrundfarbe schwarz definiert (BACKGROUND-COLOR: #000000;).

Dies führt im IE (fälschlicherweise) aber nicht dazu, dass der Bereich, der den "Punkt" im Radiobutton enthält schwarz dargestellt wird, dieser bleibt hier immer weiß. (Das lasse ich zur Not auch als interpretationsfähig durchgehen, habe mir nicht die Mühe gemacht, die entsprechenden RFCs rauszusuchen...) Opera stellt hier jedoch den Hintergrund wie im CSS angegeben schwarz dar, was es ziemlich unmöglich macht, den sehr dunklen "Punkt" (color: #332313;) zu erkennen.

Dies ließe sich durch eine kleine Korrektur im CSS schnell beheben, beispielsweise indem man, sollte man die Farbe 332313 gerne für Eingabefelder beibehalten wollen, eine helle Farbe nur für Radiobuttons definiert...
