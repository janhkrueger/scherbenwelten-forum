Gredalor (RIP) @Slim
Cache des FireFox abschalten:
Oft Verbraucht der Firefox mehr Speicherplatz als einem lieb ist. Abhilfe dabei schafft die Begrenzung des Caches, oder gleich die Abschaltung von diesem.

Ein Limit festlegen:

* Gebt in der Adressleiste von Firefox about:config ein
* Fügt eine neue Integer Variable hinzu Namens browser.cache.memory.capacity
* Gebt nun eurer neuen Variable einen Wert und beachtet, dass dieser in KB angegeben wird. 4096 wäre eine übliche Wahl.

Wollt ihr gleich den ganzen Cache abschalten solltet ihr nicht browser.cache.memory.capacity auf 0 setzen sondern eine neue boolsche Variable Namens browser.cache.memory.enable hinzufügen und sie auf false setzen.
