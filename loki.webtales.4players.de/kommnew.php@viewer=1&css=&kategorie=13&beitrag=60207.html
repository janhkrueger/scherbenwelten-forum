Mini-Änderung im CSS
Lugenmaul (RIP) Z.Zt. sieht das CSS für die Inputfelder so aus:

input {
background-image: url(m.jpg);
font-family: Arial;
font-size:12px;
border: 1px solid #777777;
margin-left: 0px;
color: #332313;
background-color: #000000;
}

Das führt bei einigen Browsern dazu, die Hintergrundbilder bei Inputfeldern nicht unterstützen, dass alle Inputfelder dunklegraue Schrift auf schwarzen Grund haben - was bspw. das Handeln etwas mühselig macht. ;-)

Deshalb mein Vorschlag:
Hintergrundfarbe nich #000000 sondern ein, dem Bild "m.jpg" entsprechendes Beige.
Und der Einfachheit halber habe ich es schon einmal rausgesucht: #DBBF80

Und hier das abgeänderte StyleSheet:

input {
background-image: url(m.jpg);
font-family: Arial;
font-size:12px;
border: 1px solid #777777;
margin-left: 0px;
color: #332313;
background-color: #DBBF80;
}
