DoomsdayProphet (RIP) Man bekommt eine Systemnachricht, wenn eine Stadt angelaufen wird. Dadurch kann man ohne großes Suchen sehr gut abschätzen, wo die sich gerade befindet.

Die Berechnung des Entfernungsfaktors (und damit der Multiplikator für den Ertrag) ist nicht ganz einfach.

Beispiele:

A -> B -> A = Gesamtlänge A -> B

A -> B -> C -> A = Gesamtlänge A - > B -> C

usw.
