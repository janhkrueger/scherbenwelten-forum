Iga Pelogran (RIP) So ist es momentan:
Städte verlieren Einwohner wenn der Platz nicht ausreicht.
So soll es werden:
1. Städte wachsen bis ans Maximum heran, verlieren aber keine Einwohner, wenn das Max überschritten wird.
2. 10% Chance, dass ein Unglück passiert (je zur Hauptauswertung), dann verliert die Stadt 10% Einwohner
3. Diese Katastrophen/Plagen werden von den Absieben (bei Ketzern von den Sieben) geschickt. Eine Immunität gibt es bei Bau der gegnerische Kultstätte.

Beispiel: Eine Stadt mit allen Tempeln ist immun.
Es gibt folgende Katastrophen:
Flutwelle, geschickt von Margintanor (Immunität für Bendurtempelbesitzer)
Pest, geschickt von Olimanir (Immu für Turatempelbesitzer)
und so weiter

Flutwelle, geschickt von Bendur (Immu für Margi-Kultstättenbesitzer)

((Besitzer=BM der Standortstadt))
