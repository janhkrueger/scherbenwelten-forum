_Sonstiges - Routen legen über das Meer
Berenike
So ist es momentan:
Als "nicht Seefahrerin" kann ich keine Routen über das Meer legen, auch wenn ich durch einen Seefahrer gefahren werde.
So soll es werden:
Es sollte möglich sein eine Route zu legen, wenn mann von einem Seefahrer mitgenommen wird.
Vorteile für das Spiel:
Bessere Anbindungen zwischen den Inseln -
Mehr Zusammenarbeit zwischen den Mitspielern
Nachteile für das Spiel:
Evtl weniger Routen auf dem Landwege
Mögliche Problematiken:
