Grafiken - Indiduelle Innenstadt-Gestaltung
Iga Pelogran (RIP) So ist es momentan:
Alle Städte und Stadtgebäude haben ein Einheits-Design und es gibt keine Gestaltungsmöglichkeiten oder Anlehnung an Rassen.
So soll es werden:
Jeder Bürgermeister bekommt ein Extra-Menü in den Stadtoptionen. Hier kann für jedes Gebäude eine Alternativgrafik hinterlegt werden und eine Checkbox "Standard verwenden".
Die hinterlegen Bilder unterstehen natürlich strengen Regeln:
- Kein Extreme Abweichung der original Grafiken
- Keine Verwirrung (Dass ein Gebäude aussieht wie das andere)
- Gebäudetyp (Arena, Kaserne etc) muss DEUTLICH erkennbar sein
- Die Bilder müssen vonder SL freigegeben werden (wie Nationskleidung z.B.)
Vorteile für das Spiel:
Mehr Gestaltungmöglichkeiten
Anpassung der Stadtgrafiken an die vorherrschende Rasse

Mann kann auch nur kleine Änderungen machen: Elfen: Alle Gebäude mit Bäumen umgeben, Andere Dachfarben oder sowas...
