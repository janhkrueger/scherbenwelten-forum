Eine ruinierte Stadt aufpäppeln, heißt täglich Waren reinschmeissen, aber nur kleine Mengen von jedem Gut, das eine stadt braucht. Hochpäppeln bis 1500 Einwohner, dann kann man sich zur Wahl stellen. Es erscheinbt ein Button "Rathaus" auf der Menueseite.
Wenn die Wahlen laufen, fleissig weiter versorgen, nur diesmal wird der tägliche Bedarf im Stadtmenue angezeit. Wenn ich es richtig mitbekommen habe, können in der Zeit maximal 250 Einheiten je Warengruppe eingelagert werden.

Wenn jedoch eine Stadt crashed und man möchte Neuwahlen (BM ist verschwunden, z.B.) dann muß man die Stadt so lange crashen lassen, bis sie ruinierte Stadt wird. Dann kann man sie wieder aufpäppeln. (siehe oben) 		15.09.10 21:17
