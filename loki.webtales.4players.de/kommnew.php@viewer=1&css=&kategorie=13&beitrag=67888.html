Forum/Diplo/Chat - Checkbox statt Pulldown (1 )
Marduc (RIP) So ist es momentan:
Zur Auswahl viele Optionen, z.b. Waren an eine Stadt verkaufen oder Produktion einstellen, wird über ein Pulldownmenü gelößt.
So soll es werden:
Wenn die Möglichkeiten eine gewisse Anzahl z.B. 3 nicht übersteigt so sollte die Auswahl durch eine Checkbox ersetzt werden.
Vorteile für das Spiel:
Der Spieler sieht sofort alle Auswahlmöglichkeiten und erspart sich ein Klick.
Nachteile für das Spiel:
Einige Spieler müssten sich an dieses System erst einmal gewöhnen.
Mögliche Problematiken:
Der Aufwand ist zu Beginn der Realiesierung recht hoch und viele Funktionen müssten angefasst werden.
Sonstiges:
