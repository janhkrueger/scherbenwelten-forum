Kultstätte
In einer Kultstätte kann der zugehörigen Gottheit gespendet werden. 10% der gespendeten Waren verbleiben dabei im Lager der Kultstätte und können vom Erbauer weiter genutzt werden.

Die Glaubensrichtung einer Kultstätte kann nach Bau gesetzt werden, allerdings nur innerhalb des gleichen Pantheons wie der Spieler selbst.

Nur Anhänger von Dämonen können eine Kultstätte einem Dämon weihen.

Das Weihen einer Kultstätte zu einer anderen als der eigenen Glaubensrichtung beinhaltet den Verlust allen Glaubens.


Vor allem auf den letzten Absatz sollte man achten, bei Umweihung einer Kultstätte! 		11.11.13 18:22
