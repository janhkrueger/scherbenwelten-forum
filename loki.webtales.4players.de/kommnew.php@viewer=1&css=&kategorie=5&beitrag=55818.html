DeLaCroix (RIP) Und wie erreicht man, dass ein Satz pro Person nur einmal gesagt wird? 14.07.05 16:03
Vincent von Drachenstein (RIP) Siehe dort: http://www.scherbenwelten.de/help.php?SEARCH=gerede&id=248

Mehrfach - Ist diese Box markiert, dann kann der Spieler mehrfach den gleichen Satz auslösen. Standardmässig ist diese Option aktiviert. D.h. jedesmal wenn jemand "Hallo" sagt wird diese Zeile aktiv, egal wie oft dieser Spiele die Zeile sagt. Meist wird diese Option deaktiviert wenn ein bestimmter Effekt eingestellt wird, wie zum Beispiel der Erhalt von Gegenständen, da dieser pro Spieler nur einmal erfolgen soll.

Das müsste die Option sein... wen du die deaktivierst sollte der Satz pro Spieler nur einmal funktionieren - glaube ich *g*
