System/Engine/Technisches - Automatische KE bei Katapult (10 )
Patriarch der III. (RIP) So ist es momentan:
Zerstörer, die Gebäude katapultieren, werden nicht automatisch mit einer KE bedacht.
So soll es werden:
Wenn jemand ein Gebäude mit einem Katapult angreift sollte er automatisch in die KE-Liste des Opfer aufgenommen werden.
Vorteile für das Spiel:
Ein Wachdienst, der auf einem eigenen Gebäude gelegt wurde, würde bei einem massiven Überraschungsangriff von Unbekannten dann auch wirklich greifen und nicht tatenlos zusehen. Also können Gebäude besser durch einen WD besser geschützt werden während man offline ist was ja Sinn und Zweck von Wachdiensten und Raubüberfällen ist - Handeln, Absicherung während der Spieler nicht vorm PC hängt bzw. hängen kann. Zweitens, Unbekannte, die plötzlich ein eigenes Gebäude mit nem Katapult angreifen will man ja so oder so sofort auf der KE-Liste haben. Diese manuelle Arbeit könnte man sich dadurch sparen.
Nachteile für das Spiel:
Ich seh keine ;-)
Mögliche Problematiken:
Manche Gebäudezerstörer habens dadurch potenziell etwas schwerer, sofern sie vom Opfer zuvor nicht eh bereits schon mit KEs bedacht wurden.
