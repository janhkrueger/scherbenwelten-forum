Forum/Diplo/Chat - Verlinkung von Wappen (5 )
Balëbréthilion Târmundor (RIP) So ist es momentan:
Man kann ein Wappen verlinken.
So soll es werden:
Man kann 2 Links für Wappen angeben.
1) Wird in der Selbstdarstellung und unter Stellung angezeigt
2) Wappen fürs Forum

Gibt man bei Wappen 2 keinen Link an wird automatisch bei beiden der selbe Link übernommen.
Vorteile für das Spiel:
Durch die extreme Verkleinerung der Wappen fürs Forum kann man ein Wappen nicht für beide Ansichten optimieren, was wünschenswert wäre.
Stattdessen kann man bis auf wenige Ausnahmen ein Wappen nur entwder fürs Forum gut machen oder für den dunklen Hintergrund der Selbstdarstellung
Nachteile für das Spiel:
Etwas mehr Datenabfragung, aber denke das ist verschwindend gering.
Mögliche Problematiken:
Keine, denke die Einführung wäre sehr unproblematisch.
Sonstiges:
