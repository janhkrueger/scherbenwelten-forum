So ist es momentan:
Diplos können nur von dem jeweiligen Charakter versendet werden.
So soll es werden:
Diplos sollen von den Anhängern der bösen Konvente via verdeckten Namen versendet werden können.
Vorteile für das Spiel:
bessere RPG - Möglichkeiten
Nachteile für das Spiel:
Ausnutzung dieses "Anonymen" Statuses
Mögliche Problematiken:
Ausnutzung dieses "Anonymen" Statuses
Sonstiges:
Um Träume via systemnachricht zu senden oder eben mit dem verdeckten Namen wäre eine solche funktion wichtig fürs RPG der bösen Konvente.
Mir hängt viel an dieser Funktion, doch weiß ich auch über die gefahr der ausnutzung.
Eine weitere möglichkeit wäre gegebenenfalls diese Nachrichten an den Support zu schicken auf das er sie selbst als Systemnachricht losschickt.
Doch das dauert immer eine gewisse Zeit und würde den jetzt schon stark belastetet Support noch mehr belasten.

Nun die Idee ansich finde ich ganz Gut, jedoch:
* Eigenes Postfach für Nachrichten an das Paktiererpseudonym, um vermischungen bei zuvielen Nachrichten (wie BMs) zu vermeiden.
* Eben genau die Möglichkeit jemanden anzuschreiben unter dem Pseudonym
* Eigene Liste (ähnlich Freund/Feind) wer dem Paktiererpseudonym schreiben kann (wegen Spamgefährdung, das Paktierer sind doof is mir schon im Forum zuviel)
* Jemanden anschreiben heisst er wird automatisch auf diese Liste gesetzt
* Möglichkeit für Support die Benutzung für den Paktierer bei Ausnutzung zu sperren

Problematik: sehr aufwendig mit diesen Einschränkungen
