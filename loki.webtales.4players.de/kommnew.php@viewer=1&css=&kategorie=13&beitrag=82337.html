Skills/Berufe/Rassen - XP für Seefahrt (1 )
Der schwarze Ritter (RIP) So ist es momentan:
Der Seemann fährt mit seinem Schiff herum, meistens um anderen einen Gefallen zu tun, er bringt damit seinen Schiffahrtsskill hoch, aber er erhält dafür keine Erfahrungspunkte XP. Das Problem der meisten Seefahrer ist es, daß sie meistens unterwegs sind, dadurch einen hohen Seefahrtsskill erlangen, aber die Basis nicht erhöhen können, weil sie keine XP dafür bekommen.
So soll es werden:
So wie andere Berufe auch, z.b. der Bauer beim Getreide ernten, soll auch der Seemann für seine Schifffahrt XP bekommen. Ich denke das wäre diesen Leuten gegenüber fair, da sie sich oft und viel zum Wohle anderer einsetzen.
