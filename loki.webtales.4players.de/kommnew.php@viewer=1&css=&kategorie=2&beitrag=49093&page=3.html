Vorondil Coldleaf (RIP) ich war an der anderen seite des tunnels...
Ihr betretet eine große Höhle. In dem Gang nach Norden hört ihr das Tropfen von Wasser.
Eine wacklig aussehende Holztreppe scheint nach oben zu führen. In der Mitte des Raumes seht ihr die Überreste einer großen Feuerstelle.
Diverse abgenagte Knochen, sowie einige leere Fässer liegen um die Feuerstelle verteilt. Ihr entdeckt die Überreste von schimmligem Brot.

Um die Feuerstelle verteilt sind zahlreiche, verlassene Schlafstätten. Fast scheint es euch, als hätte hier eine Kompanie längere Zeit übernachtet.

Von den Wänden hängen zwei große Banner, auf denen auf schwarzem Grund eine siebenfingrige Hand dargestellt ist. Dazwischen ist mit einer rötlich braunen Farbe eine Karte der Lagerinseln skizziert. An sieben Stellen ist eine Art Turm eingezeichnet. Neben jedem dieser Türme ist eine Rune eingezeichnet. Vielleicht ein Klanzeichen?

Über der Karte sind Runen angebracht. Nach kurzer Rücksprache könnt ihr sie entziffern. In blutroten Zeichen steht dort „Ernte“

Auf einem Tisch in der Ecke ist aus Sand eine Insel modelliert. Sie scheint einem Beil zu ähneln. Drei Städte sind eingezeichnet, eine am Ende des Schafts. Eine an der Klinge, eine an der Rückseite der Klinge. Zwei Anleger scheinen eingezeichnet. Eine im Süden, eine im Norden. Zwei große Pfeile deuten auf diese Anleger. Daneben ist eine Krake vor einer Gewitterwolke abgebildet. Daneben seht ihr im Norden seht ihr eine stilisierte Klaue, im Süden eine Peitsche.

eigenartig... könnten diese türme etwa die städte sein wo die orks angreifen werden? oder etwa die hauptstützpunkte der orks? diese insel in form einer axt... es könnte ihre hauptiinsel sein von wo aus sie agieren...
