Parrakus (RIP) Hab mal einen Link zu einem Online-Game rausgekramt.

http://www.deathball.net/notpron/startd.htm

notpron ist ein Rätsel das immer aus folgenden Dingen besteht:

1. einem Bild (mit der jeweiligen nummer des levels)
2. der Url des rätsels (die ist oft ganz wichtig)
3. und dem Source Code des jeweiligen rätsels (Quelltext der Seite)


Jede dieser oben genannten Dinge kann Infos für die Lösung des Rätsels haben.

Die Lösung ist entweder:

1. eine Änderung in der Url (immer das letzte wort vor .htm(l)
2. oder es gibt ein Eingabefenster für einen Benutzernamen oder ein Passwort

Manchmal benötigt Ihr auch ein externes Programm (z.B. um Bilder anzusehen etc)
