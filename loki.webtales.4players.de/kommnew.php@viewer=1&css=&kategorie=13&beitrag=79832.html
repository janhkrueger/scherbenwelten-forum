Dráthir Dálhar (RIP) So ist es momentan:
Wenn man in der normalen Ansicht einen Keller betritt, sieht man lediglich den nackten Keller.
Stellt man im Keller die Iso ansicht ein, kann man, so vorhanden, Kamin,Betten etc. erst sehen.
So soll es werden:
Wie in Wohnhäusern soll auch in Kellern die Automatische Iso ansicht aktiviert werden.
Vorteile für das Spiel:
Das gesamtbild wird verschönert.
RPG potential.
Der Keller ist nicht mehr einfach nur ein muss um ein größeres Gebäude zu bekommen.
Möbel wie Kamin, betten usw. gewinnen an wert da sie mehr gefragt sind.
Nachteile für das Spiel:
hmm, euch werden sicher mehr einfallen.
Mir dagegen fällt nur einer ein.

Manche leben auf ihren Schiffen, doch scheinbar kaum einer in seinem Keller, welchen man ebenso gut als Wohnstube deklarieren könnte.

Die, die schonmal bei meinem Gebäude waren, wissen das ich dort eine Wohnung eingerichtet habe, mit Gästebett Kamin, eigenem Schlafzimmer, eben was man in einem Wohnhaus auch bauen könnte.
