RPG - Glaubenskrieger Richtigstellung (11 )
Williams Horlus (RIP) So ist es momentan:
Wird man von einem Konvent des Lichtes verketzert so erscheinen Glaubenskrieger vor seiner Tür. Dies ist in Ordnung.

Doch Glaubenskrieger greifen auch Tempels an und sogar Tempels von seinem eignem Glauben.
So soll es werden:
Glaubenskrieger greifen alle Gebäude an ausser den Tempel seines eignen Glaubens.
Vorteile für das Spiel:
So bekämpft der Glauben sich nicht selbst. Sieht komisch aus, wenn auf einmal Glaubenskrieger seine eigne Tempel vom Glauben zerstören.

Jeder Anhänger seines Glauben sieht nicht gerne, wenn ein Tempel seines Glaubens zerstört wird und wenn er noch vom eignen Gott zerstört wird, was soll man da denken?
Nachteile für das Spiel:
Im Grunde keine ist eine RP Änderung.
