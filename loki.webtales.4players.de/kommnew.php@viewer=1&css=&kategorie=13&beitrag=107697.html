Braniag (RIP) So ist es momentan:
Wird ein Spieler gesperrt und kann daher seinen Charakter nicht mehr kontrollieren oder benutzen, wird das von der Spielleitung nicht dokumentiert.
Wird dieser Spieler dann Opfer, wird das von Teilen der Spielerschaft als Ausnutzung der Sperre und somit als nicht fair beurteilt.
So soll es werden:
1. Gesperrte Spieler werden in einer Liste geführt und von der Spielleitung öffentlich gemacht.
2. Gebäude und Städte des gesperrten Spielers´ Charakter werden mit dem Zusatz (GESPERRT) versehen.
3. Die Spielleitung stellt fest ob die Anzeige dem Anzeiger einen Nutzen verschafft.
4. Die Spielleitung achtet darauf, dass keine Aktionen gegen diesen Spieler stattfinden, solange die Sperre dauert.
5. Wird der Spieler aus dem Spiel entfernt, wird dieser Fakt von der Spielleitung bekannt gegeben.
Vorteile für das Spiel:
Mehr Fairness gegenüber Spielern, die zu unrecht oder auf Grund von Aboproblemen gesperrt wurden.
Nachteile für das Spiel:
Später überführte Benutzer von Multiaccounts werden azunächst geschützt, aber hier gilt die Unschuldsvermutung bis zur Überführung.
Mögliche Problematiken:
Angegriffene Spieler könnten versuchen eine Anzeige zu erwirken, die sie dann sperrt und damit einen Krieg verhindern.
Ich denke aber, dass solche Spieler leicht dieser Tatsache zu überführen sind. Die Repressalien Ingame kann sich dann ja jeder vorstellen.
