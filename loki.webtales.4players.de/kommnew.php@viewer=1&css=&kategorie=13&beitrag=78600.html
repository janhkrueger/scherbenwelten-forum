Gebäude/Rezepte/Produktion - Diplo beim Prod. von Gegenstäde
Mahtan Míriel (RIP) So ist es momentan:
Wenn ein Gegenstand produziert wurde kommt keine Nachricht, nur wenn man "zufällig" auf seinen Magierturm oder seine Schmiede klickt, kriegt man das dann mit.
Das Problem dabei ist, das dadurch (mir zumindest) immer 2-5 Schichten verloren gehen, wenn ich mal nicht dran denke.
So soll es werden:
Einfach eine kurze Diplo: "Dein Gebäude auf (xxx/xxx) hat einen Gegenstand produziert".
So wäre es am einfachsten.
Optimal wär allerdings: "Deine Schmiede (xxx/xxx) hat einen Zweihänder produziert"...zum Beispiel.
Vorteile für das Spiel:
Man ärgert sich weniger und spart vorallem Zeit beim Produzieren von verschiedenen Gegenständen...
Nachteile für das Spiel:
ich sehe keine
Mögliche Problematiken:
Traffic, für einige evtl. "Diplomüll".
Sonstiges:
Es gibt nun wirklich schon so viele unsinnige Diplos wie "Deine Karawane durchquert Stadt x", dass ein sinnvoller mehr nicht schaden kann. 		23.02.06 17:33
