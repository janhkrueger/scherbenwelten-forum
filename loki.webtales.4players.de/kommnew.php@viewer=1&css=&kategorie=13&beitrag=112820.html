Forum/Diplo/Chat - Ausblendung von OOC Kommentaren
Heindal (RIP) So ist es momentan:
OOC Kommentare werden im Chat/Diplonachricht angezeigt und führen dazu das Regeln verletzt.
So soll es werden:
OOC Kommentare die explizit in Doppel-Klammern gesetzt werden, werden ausgeblendet und können auf Wunsch angezeigt werden. Durch klick bsp. Dies ist vergleichbar mit Questspoilern.
Vorteile für das Spiel:
Weniger Aufwand bei der Ahndung von ooc Kommentaren im Chat, mehr Möglichkeiten im Chat.
Mehr Freiheit für Chatter, die ohnehin nur einen kleinen Teil des Spiels aus machen.
Nachteile für das Spiel:
OOC Flamen könnte zur Gewohnheit werden.

Mögliche Problematiken:
Technische Realisierung. Ich kann mir vorstellen, dass es schwer zu realisieren ist, es gibt auch in Foren Automatiken die es erlauben "Spoiler" einzufügen die ausgeblendet werden. Wie und ob das im Chat möglich ist, entzieht sich allerdings meiner Kenntnis.
