So ist es momentan:
In den Foren von RP Gruppen gibt es keinerlei Möglichkeiten zur Sortierung und Suche.
So soll es werden:
RP Foren erhalten Möglichkeiten zur Sortierung

- Durch den Ausbau von Ablagefächer können Unterforen zur Organisation zu erstellt werden.
- Der Ausbau von Studierzimmern ermöglicht es Moderatoren auszuwählen die Beiträge im Forum verschieben können.
Zusätzlich können Beiträge angepinnt werden.

Zusätzlich erhalten Foren eine Möglichkeit zur Suche im Forum wie vorgeschlagen bei:
http://www.scherbenwelten.de/kommnew.php?viewer=&css=&gesucht=1&kategorie=13&beitrag=112010
Vorteile für das Spiel:
- Man braucht alte Beiträge im Forum nicht per Hand suchen.
- Bessere Kommunikation mit IG Mitteln
- Minimaler Aufwand die Suchfunktion zu aktivieren
