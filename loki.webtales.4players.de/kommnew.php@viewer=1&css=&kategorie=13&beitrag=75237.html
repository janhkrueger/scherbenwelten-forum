Forum/Diplo/Chat - Feste Queste in der Scherbe (8 )
Marduc (RIP) So ist es momentan:
Einige wenige Queste können auf der Scherbe ständig abgelaufen und gelößt werden (Robenquest/Wichtekarte).
So soll es werden:
Eine große Anzahl an festen Queste sollen über die Scherbe verteilt werden.
Vorteile für das Spiel:
Neue sowie ältere Spieler die den Quest noch noch nicht gelößt haben können neben ihrer Produktion RPG nachgehen.
Nachteile für das Spiel:
Feste Regelabläufe können den Quest schnell langweilig und Lösung im Forum erlesen werden.
Mögliche Problematiken:
Powergamer reisen alles sofort ab, Diebe finden sich bei den Questorten ein und was sonst der Standardeinwand ist.
Sonstiges:
Durch kleine Gimiks bzw. durch leichte Aufwertung von Gegenständen können diese Queste nicht nur Powergamers Spass machen.

Hier mal ein Beispiel für ein Seefahrerquest:

Der Spieler benötigt zum lösen der Quest eine Barke.
Die Aufgaben die dem Spieler gestellt werden sind mehr zufälle zusammengesetzt, so muss der Spieler Waren von einer Stadt zu einer anderen über den Seeweg bringen, auf einer Koordinate ein Hai erlegen, mindestens x Fische gefangen haben ( Fischenskilldiv. zwischen Questannahme und Questende) usw.
Als Belohnung winkt dem Spieler im Hafen eine Schiffsaufwertung der eigenen Barke.
Erhöhung der Reichweite von 50 und Erhöhung der Tragkraft von 100.
