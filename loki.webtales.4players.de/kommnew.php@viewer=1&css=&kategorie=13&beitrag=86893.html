Leta (RIP) Es gibt nicht so Unmengen an Kunstobjekten, das man damit die Landschaft vollmüllen könnte. Ich versuche mal zusammenzufassen wie eine mögliche Lösung sein kann:








-Kunstobjekte außerhalb von Innenstädten errichtet werden.
-Kunstobjekte können in Innenstädten mit Bauerlaubnis errichtet werden.
-Auf dem Feld eines Kunstobjektes können auch andere Gebäude errichtet werden, allerdings kann man das Kunstobjekt danach immer noch einsammeln.
-Das Kunstobjekt hat nach wie vor keinen Zustand.
-Man kann beim Kunstobjekt eine Geschichte hinterlegen, wie eine Gebäudebeschreibung. Es sollte dann auch in der Gebäudeliste erscheinen (unter Übersicht).
-Beim errichten eines Kunstobjekts zieht sich der Server automatisch die beiden angegeben Grafiken. Damit wäre Mißbrauch später eindeutig nachweisbar.
