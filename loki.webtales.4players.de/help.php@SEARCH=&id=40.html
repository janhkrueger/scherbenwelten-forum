Städte, Besonderheiten

  • Sinken die Einwohner einer Stadt unter 500, wird der Bürgermeister automatisch abgesetzt. Sein Einfluss sinkt auf 0, alle anderen um die Hälfte.

  • Sinken die Einwohner einer Stadt 10 Tage in Folge, wird der Bürgermeister automatisch abgesetzt. Sein Einfluss sinkt auf 0, alle anderen um die Hälfte.

  • Das Erscheinungsbild (Grafik) einer Stadt ändert sich entsprechend der Hauptrasse der Einwohner.
    Bei mindestens 60 % einer Rasse in einer Stadt wird ein entsprechendes Stadtbild generiert.
