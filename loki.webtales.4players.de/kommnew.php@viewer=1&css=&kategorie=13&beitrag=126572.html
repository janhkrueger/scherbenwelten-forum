Kampf/Armeen/Monster - Ansicht Belagert
Iga Pelogran (RIP) So ist es momentan:
Man kann den Status von Gebäuden über die Überischt herausfinden. Ist das Gebäude belagert, so kann man es zwar nicht einstellen, aber man weiss dass es belagert wird und kann es räumen/meiden.
So soll es werden:
Variante A: Städte bekommen einen Status in der Stadtübersicht Frei/Belagert.
Variante B: Städte können bei Belagerung nicht eingestellt werden, sowie gebäude.
Variante C: Beim Versuch eine Handelsroute zu nutzen wird eine Warnung ausgesprochen. ("Die Händler haben erfahren, dass die Stadt belagert wird. Wollt Ihr trotzdem dorthin?")
Vorteile für das Spiel:
- Man weiss, ob eine Stadt frei ist, oder ob man mit GKs, Untoten und evtl. anderen Monstern rechnen muss
- Die 50 Reisewarnungen sind aus dem Regiochat raus
Nachteile für das Spiel:
Weniger Überraschung
