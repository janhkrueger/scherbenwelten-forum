Marduc (RIP) Eine andere Variante wäre ein Forenbereich einzuführen. Dort werden Regelverstöße von Usern gemeldet.
(Als Beispiel: 30K für Schild und 30K für Avatar)
Der Moderator dieses Forenbereiches kontrolliert die jeweiligen Regelverstöße und mahnt den Spieler durch eine Diplo an.
Nach einer Frist von z.B. 1 RL-Woche wird erneut geprüft, bzw. Aronius führt eine Funktion ein, die nach Mahnung automatisch prüft.
Nach Ablauf der Funktion/Frist, wird das Userprofil auf Standard geändert, d.h. der Schild und der Arvatar wird auf Standard geändert.

Der User bekommt die Möglichkeit im passendem Thread sich zu melden und dort seine neuen Grafiken vorstellen.
Der Moderator kann dann die Funktion wieder zurücksetzten bzw. den User wieder für Customgrafiken Freischalten.

Nachteil dieser Variante:
- hoher Pflegeaufwand
- massiver Eingriff in eine freie Entfaltung
- diktieren von Regelwerken
- auch hier muss Aronius eine Funktion einführen 		31.05.05 13:25
