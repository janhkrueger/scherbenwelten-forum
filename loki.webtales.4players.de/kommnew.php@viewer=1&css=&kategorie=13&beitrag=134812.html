_Sonstiges - Übergabe ohne Namen (12 )
Markus Feuernacht
So ist es momentan:
Bei jeder Übergabe steht dabei wer die Übergabe getätigt hat
So soll es werden:
Es soll eine Option geben mit der man den Übergabe Namen verschleiern kann.
Vorteile für das Spiel:
Die Interaktion der Spieler kann bereichert werden. So könnte man einem Feind einen Gegenstand versteck zustecken der z.B. Grosser Heiltank heißt aber schaden verursacht. Oder aber auch das RP bereichern wenn "kein Nachweis" über die Herkunft eines Gegenstandes gegeben werden kann ohne die SL für diese Übergabe ein beziehen zu müssen.
Nachteile für das Spiel:
Es gäbe eine weitere Option die man bedenken muss und die irgend wo wählbar sein muss.
Mögliche Problematiken:
Die Implementierung bzw das Binden an einen Skill wenn das entdecken solcher Zustecker auch Skill abhängig ist.
Sonstiges:
Ich befürchte viele sehen das als unnützen Quatsch, aber es ergäbe einige neue und auch Lustige Möglichkeiten für das Spiel denke ich.
