Gnisha (RIP) So ist es momentan:
Städte sind im Bereich von ca 50k am effektivsten, über 100k Bewohner fangen sie an Verlust zu machen. Dieses hat zur Folge, dass nahzu alle Städte in diesem Bereich gehalten werden. Dieses hat logischerweise massive negative Einflüsse auf die Wirtschaft, es findet kaum Wachstum stadt, da dieses unprofitabel wäre.
So soll es werden:
Eine Anpassung der Steuerberechnungsfunktion, dass sobald es mehere Bevölkerungsschichten in einer Stadt gibt, die Steuern ansteigen. Ein Kaufmann zahlt mehr Steuern als ein einfacher Arbeiter, und nur er braucht auch ein Theater etc... So könnte eine Waage zwischen Meerausgaben für Bücher, Schmuck und Steuern etc getroffen werden.

Vorteile für das Spiel:
- Ankurbelung der Wirtschaft duch eine bedeutend größere Nachfrage an Ressourcen
- Stärkeres Konkurrenzverhalten, da dann vermutlich nicht mehr alle Städe überleben könnten, wenn es profitabler wäre, eine große anstatt vieler kleine Städte zu unterhalten.
-Wieder Sinn zur Spezialisierung duch einen Bedeutend höheren Verbrauch
Nachteile für das Spiel:
- sehe ich direkt keine
Mögliche Problematiken:
- es sollte lediglich nur eine Anpassung des Steuerskriptes erfolgen müssen
