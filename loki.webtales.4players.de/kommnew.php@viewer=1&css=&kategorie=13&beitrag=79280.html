Forum/Diplo/Chat - Rechte für Verkünder im Konventsforum (12 )
Meklun Wellenreiter (RIP) So ist es momentan:
Beiträge in den Konventsforen können nur vom jeweiligen Erfasser mit dem Fisch als abgeschlossen gekennzeichnet werden.
So soll es werden:
Der jeweilige Verkünder des Konventes sollte parallell zum Erfasser eines Beitrages das Recht besitzen, einen Thread zu schliessen.
Vorteile für das Spiel:
Somit könnten auch Beiträge wie Verkündungen, Verketzerungen usw., die ja eigentlich "niemandem" gehören, geschlossen werden. Ausserdem könnten Beiträge von Personen, die sich nicht mehr im Konvent befinden, geschlossen werden.
Nachteile für das Spiel:
keine bekannt
Mögliche Problematiken:
Es werden zu viele Beiträge vom Verkünder geschlossen....
Sonstiges:
Evtl. wäre auch das wieder Eröffnen von Beiträgen anzudenken. 		08.03.06 10:02
