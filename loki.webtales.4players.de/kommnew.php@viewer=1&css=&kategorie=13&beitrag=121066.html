Gebäude/Rezepte/Produktion - Waffen weihen/Brunnen in Kulti (1 )
Iga Pelogran (RIP) So ist es momentan:
In einer Kultstätte kann nur geopfert werden. Großausbauten an einer Kultstätte machen Enginetechnisch keinen Sinn. Rp mäßig auch nur dann, wenn man im Keller wohnt oder eine Bibliothek oder so aufstellen will.
So soll es werden:
Man bekommt nach dem 1. (oder evtl 2.) GA die Möglichkeit Brunnen zu errichten (wie Knechte in der Schmiede).

Diese Brunnen generieren je 1 Weihwasser pro Tag (Wie Antragspunkte im KOnvent). (Angesammelt werden können max. 100 Weihwasser)

Für 50 Weihwasser kann man eine Waffe weihen. Diese bekommt neben ihrem Zustand das Attribut "geweiht".

Geweihte Waffen machen 10% (?) mehr Schaden gegen Untote.

Bei Absiebentempel könnte man evtl. anstatt Brunnen irgendwas anderes aufstellen (Kessel) und könnte Gegenstände verfluchen (was auch immer dann passieren mag.
Vorteile für das Spiel:
- Aufwertung der Kultstätte
- Mehr Interaktion ("Hey kannst Du mir meine Waffe weihen? - "Macht 5k GS")
- Vielfalt
Nachteile für das Spiel:
- je nachdem was Paktis dann bekommen könnten Vor/Nachteil für diese
- Kultstättenbesitzer bevorzugt
Mögliche Problematiken:
- Attribute an Waffen zu ketten könnte durchaus schwer programmierbar werden.
Sonstiges:
Fände ich Spannend, da auch kein PvP-Balancing angetastet wird, sondern nur PvM.

Evtl. so denke ich kann man die Werte auch so anpassen:
Eine nicht geweihte Waffen macht imo x Schaden.
Nach der änderung:
Eine nicht geweihte Waffe macht x-5% Schaden (resistenz der Untoten, wirkungslosigkeit der Waffe)
Eine geweihte Waffe macht x+5% Schaden. 		28.12.09 8:49
