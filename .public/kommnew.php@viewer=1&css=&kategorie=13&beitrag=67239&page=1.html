Habe nicht alles gelesen, aber man koennte das mit den PP auch staffeln.

Jetzt NUR als Beispiel:
von 0 bis 100k EW -> je 5k = 1 PP
von 100 bis 500 EW -> je 10k = 1 PP
ab 500 EW -> je 20k = 1 PP


Also eine 200k EW Stadt =
100.000 / 5.000 = 20 PP
+ 100.000 / 10.000 = 10 PP
= 30 PP


Eine 300k EW Stadt =
100.000 / 5.000 = 20 PP
+ 400.000 / 10.000 = 40 PP
+ 100.000 / 20.000 = 5 PP
= 65 PP

Ich denke, das waere vielleicht eine Alternative.
