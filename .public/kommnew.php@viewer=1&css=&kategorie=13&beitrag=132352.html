So ist es momentan:
Wenn man z.B. Fuhrwerke unter den Chars in der Gruppe verteilen will, muss man das an jeden einzeln übergeben.
So soll es werden:
Statt hinter jedem Gegenstandstyp einen eigenen OK-Button zu haben, einen OK-Button für alle eingestellten Übergaben.
Vorteile für das Spiel:
Dieselbe Aktion geht mit deutlich weniger Klicks einher, ohne Entscheidungen etc. vorwegzunehmen. Serverlast nimmt ab, da das Berechnen und Ausliefern der Webseite vermieden wird (natürlich finden die DB-Transaktionen immer noch statt).
Nachteile für das Spiel:
Ich sehe keine
Mögliche Problematiken:
Das ist ein Paradigmenwechsel im Bedienkonzept, es ist damit zu rechnen dass Fehlbedienungen eine Zeit lang nach der Einführung auftreten.
