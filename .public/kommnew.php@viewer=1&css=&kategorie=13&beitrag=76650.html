Gebäude/Rezepte/Produktion - Heilung von Krankheiten (1 )
Rondrian (RIP) So ist es momentan:
Für die Heilung einer Krankheit werden nur Kräuter gebraucht. Krankheiten interessieren so gut wie niemanden, sind im höchsten Fall lästig und man wird sie schnell wieder los.
So soll es werden:
Künftig wird für jede Krankheit ein eigenes Heilmittel gebraucht. Um so ein Heilmittel herzustellen gibt es ein neues Gebäude, Apotheke, in der diese Hergestellt werden können. Jedes Heilmittel muss als Rezept gekauft werden. Als Grundzutaten kommen Kräuter ins Heilmittel und Bücher als Anleitung zur Herstellung und dann diverse andere Waren.

Skill und Produktionsmäßig sollte das ganze an die Uni angelehnt sein. Also Tagesproduktion (Heilkunde+Alchemie)/4 + 2 Heilmittel pro Tag.

Nur ein Heiler der auch das Rezept besitzt, kann das Heilmittel auch verwenden. Da sich einanderer nicht damit auskennt.

Möglicherweise eine Beschränkung der Rezepte.
Vorteile für das Spiel:
Professionelle Heiler werden wichtiger.
Krankheiten gehen vielleicht über den Lästigfaktor hinaus.
Es freuen sich alle über neue Rezepte.
Und es können jede Menge verschiedene Waren aus dem Spiel genommen werden.
Nachteile für das Spiel:
Einiges an Aufwand für den Einbau.

Vielleicht am Anfang zu wenig Heiler auf der Scherbe, da dieser Beruf ziemlich vernachlässigt ist.
Mögliche Problematiken:
Sehe da keine
Sonstiges:
Vielleicht eine kleine Möglichkeit das Heilertum aus dem Schatten zu holen. 		21.01.06 9:25
