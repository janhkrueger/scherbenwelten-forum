Gebäude/Rezepte/Produktion - Abriss von Innenstadtgebäude
Iga Pelogran (RIP) So ist es momentan:
Der Bürgermeister einer Stadt kann ohne weiteres jedes Stadt gebäude abreissen (lassen). Auch die von anderen Spielern.
So soll es werden:
Wird ein Gebäude von einem anderen Spieler abgerissen (oder soll abgerissen werden) so muss der Besitzer zustimmen. Stimmt er nicht zu so könnte man es so einbauen, dass die Stadtmiliz gegen die Gebäudebesatzung kämpft (um das Gebäude zu räumen). Dies könnte man dann wieder mit der Beliebtheit in der Stadt koppeln, der wer vertreibt schon einen guten Freund.
Vorteile für das Spiel:
- Interaktionfördernd, da man nicht so ohne weiteres ein Gebäude eines anderen in der Stadt abreissen kann.
- Kein Verlust von Waren oder Produzenten beim Abriss von innerstädtischen Gebäuden
Nachteile für das Spiel:
- Der BM hat nicht mehr soviel Macht...
Mögliche Problematiken:
- Programmierung
Sonstiges:
