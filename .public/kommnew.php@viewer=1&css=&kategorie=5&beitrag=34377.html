El Muriell Darontes (RIP) nicht ganz richtig..
Der Server merkt sich immer das letzte Feld vor deinem aktuellen Standpunkt. Das ist dein Fluchtfeld.
Ausnahmen sind:
- wecheln einer Ebene (Höhle, Innenstadt, Keller...)
- Tunnel
- gerade geflohen oder Gebäude verlassen
- Angelegt am Anleger (?)

In diesen Fällen hast du kein Fluchtfeld mehr.

Wirst du dann von Monstern angegriffen ist eine Flucht nicht möglich.
Eine weitere Möglichkeit in die Situation zu kommen ist, wenn auf dein Fluchtfeld Monster nachgerückt sind. (z.B. du stehst in einem Gebäude, eine Gruppe Ratten rückt in der Zwischenzeit auf das Feld neben das Gebäude vor. Genau auf das Feld, über das du das Gebäude betreten hast.
