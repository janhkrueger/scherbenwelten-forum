Glauben/Konvente - Gebäudestandort von Verketzerten
Iga Pelogran (RIP) So ist es momentan:
Ist jemand verketzert, so Greifen die Glaubenskrieger automatisch die Gebäude des Verketzerten an. Meistens jedoch weiss niemand wo diese Gebäude stehen.
So soll es werden:

Gebäude können nur angegriffen werden wenn der verketzernde Konvent die Koords angibt.
Vorteile für das Spiel:
- Logik, wenn die Mitgleider eines Konventes nicht wissen wo ein Gebäude steht, warum sollten es dann die Glubenskrieger wissen...?
- Paktieren wird etwas einfacher, weil man nicht unbedingt Glaubenskämpfer befürchten muss
Nachteile für das Spiel:
Paktieren wird einfacher
Mögliche Problematiken:
Es muss eine Eingabemöglichkeit geben, wo man die Koords eintragen kann
