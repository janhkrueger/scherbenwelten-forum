Iga Pelogran (RIP) So ist es momentan:
Nun, die Spielerzahlen sind Rückläufig. Das bedeutet, dass weniger Einahmen da sind. Sinkt die Spielerzahl weiter, so muss mit Konsequenzen gerechnet werden. Diese könnten sein: Hardware-Downgrades, weniger Mitarbeiter im Support, oder wenn es ganz arg wird sogar das Einstellen des Spielbetriebs. Gegenmaßnahme wäre nur eine Anhebung der monatlichen Gebühren oder Werbung einblenden. Doch bevor es zu soetwas kommt, sollte man überlegen, wie man dagegenwirken kann.
So soll es werden:
Es wird an Onlineshop errichtet auf PrePaid-Basis. Hier können Gegenstände, die man erspielen kann auch gekauft werden. Natürlich so, dass man sich nicht für eine Hand voll Geld sich extreme Vorteile erkaufen kann.
Als faustformel würde ich Vorschlagen 1ct Echtgeld entsprechen 1000 Goldstücke. Hört sich viel an aber:
1 Mittlerer Ed, könnte für 45.000 GS in diesem Shop ergo für 45ct verkauft werden. Selbst wenn jemand 45 Euro, was ein Jahresabo entspricht bezahlt, hat man grade mal 30k AP dazugewonnen! Kaufbar soll nahetu alles sein:
Gegenstände zu manuellfestgelegten Preise
Waren zum SM-Preis
Goldstücke 1ct:1000GS (10 Euro = 1M GS)

Es sollten keine Premiumgegenstände da sein ,sondern nur Dinge, die man sonst auch erspielen kann (ausser Skills)...
Vorteile für das Spiel:
- Das Spiel läuft nicht so schnell Gefahr in Finanznöte zu kommen.

- Wenn viel Genutzt, können die monatlichen Kosten gesenkt werden
Nachteile für das Spiel:
- Reiche Leute können sich dann doch extrem-Vorteile erkaufen
