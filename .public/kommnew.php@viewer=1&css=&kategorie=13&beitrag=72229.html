Glauben/Konvente - Fluchtmöglichkeiten bei Glaubenstruppen (4 )
Tharsonius (RIP) So ist es momentan:
Wenn jemand Ketzer gegen einen Gott/Dämon ist und auf Glaubenskrieger des Gott/Dämons trifft kann er weiterhin sich aus dem Kampf zurückziehen und sich z.B. heilen damit er dann die Glaubenskrieger besiegen kann die seine Gebäude belagern.
So soll es werden:
Ketzer gegen einen Gott/Dämon können sich nicht mehr zurückziehen wenn sie auf Glaubenskrieger/Untote des Konventes treffen wo sie Ketzer sind.
Vorteile für das Spiel:
Ist einfach mal realistischer das die Truppen denjenigen den sie Suchen nicht einfach gehen lassen. Außerdem besteht so auch mal die Möglichkeit das Glaubenstruppen die Leute erwischen gegen den sie ausgeschickt wurden. Die Glaubenstruppen werden so etwas aufgewertet.
Nachteile für das Spiel:
Es werden Spieler im Tempel landen, die Glaubenstruppen der Konvente werden aufgewertet. Glaubenstruppen können nicht mehr so einfach beseitigt werden.
Mögliche Problematiken:
Leute sterben? Ka, ich sehe keine Probleme.
Sonstiges:
Dadurch überlegt man es sich vielleicht in Zukunft 2 mal Glaubenstruppen anzugreifen. Bisher sind sie nur lästig, stören aber niemanden wirklich außer das sie bisschen die Produkion behindern. 		14.11.05 2:59
