Gebäude/Rezepte/Produktion - Massenwaren (194 )
Gwydion de Ville (RIP) So ist es momentan:
Massenwaren Produzenten (Holz, Fleisch, Getreide, Stein) sind regional sehr gebunden, da die Lager meist zu begrenzt sind um große Städte ausreichend zu versorgen. Deshalb muss eine Leerung der Gebäude täglich erfolgen. Daraus resultiert eine gewisse Unattraktivität für die Besitzer solcher Produktionen, da sie sehr gebunden sind.
So soll es werden:
Zwei Mögliche Varianten:

Variante 1:

Die oben genannte Massenwaren werden im Vergleich zu anderen Waren in eine andere Kategorie eingestuft, so dass nur noch 3 (evtl. sogar 5) Einheiten von Massenwaren einen Lagerplatz verbrauchen (z.B. 3000 Stein benötigen 1000 Lagerplatz).
Andere Waren sollen den Ursprünglichen Lagerplatz benötigen.

Variante 2:

Die Massenwaren in der momentanen Form werden aufgelöst. Zum Beispiel wird statt Holz Holzbretter produziert, anstatt Stein Baumaterial, Getreide wird zu Mehl und Fleisch könnte durch Fleischpakete ersetzt werden. Die Produktionsmenge wird erheblich reduziert auf so dass es in der Täglichen Produktion in etwa der einer Bäckerei entspricht. Dadurch könnte ohne eine zweite Kategorie die Lager erheblich entlastet werden.
Vorteile für das Spiel:
Massenwaren Produzenten werden entlastet und können sich "freier" über die Scherbe bewegen. Der Frust sollte sinken und somit eine solche Produktion attraktiver werden. Auch könnte dadurch evtl. die Multianzahl gesenkt werden, da diese Produktionen nicht mehr soviel Aufmerksamkeit (AP) erfordern.
Nachteile für das Spiel:
keine wirklichen (außer evtl. Punkt allgemein bei den Problematiken)
Mögliche Problematiken:
Variante 1: Probleme mit einer Kategorisierung, soltle aber nur ein kleiner engine-technischer Kniff sein. Wichtig ist halt, dass das Lager nur für diese Massenwaren angehoben wird.

Variante 2: Komplettes Auflösen einer bereits bestehende Warengruppe könnte zu erheblichen Änderungen im Spiel führen.

Allgemein: Leute mit großem Lager würden sich vielleicht beschweren?
