So ist es momentan:
Man sieht die Persönlichen Nachrichten von der ersten Seite, ohne Blätter- oder Sortierfunktion. Einzige Möglichkeit ist, die PNs zu mailen und im E-Mail-Fach zu filtern.
So soll es werden:
2 Diploverwaltungsmöglichkeiten (umschaltbar unter Optionen).

1. Diploverwaltungsmöglichkeit: Einfache Ansicht (im Zweifel, so wie es jetzt ist).

2. Diploverwaltungsmöglichkeit:


2.a) Filter „Löschen“ kann deaktiviert werden.
2.b) Button „Markierte löschen“ wird leicht rötlich.
2.c) Eigene Filter können angelegt werden:
Sonderfilter: Alles/Gesendete/Empfangene aktivierbar
Schnellfilter aktivierbar
Systemfilter: {„Betreff enthält“ } {Eingabe} (z.B. Handel, Route, Karawanen,…)
Namensfilter: {„An/Von enthält“} {Eingabe} (z.B. Abdiel, Alberix, Altan,…)
> Es entstehen anklickbare Filter, die man kombinieren kann. (Bsp. siehe Bild)

2.d) SEITENANZEIGE, man kann endlich auch einfach mal durchblättern.
Damit dort am Ende bei den meisten Spielern keine 1000 steht, muss in den Optionen angegeben werden:
[_] nach 7 Tagen werden die Nachrichten gelöscht oder
[_] nach 7 Tagen werden die Nachrichten gemailt
optional nach folgendem Filtersystem (oberster Filter gilt zuerst):
> selbst einstellbar <, z.B.
1. Betreff: Routen
2. Betreff: Handel
3. Betreff: Karawanen
4. Rest
Vorteile für das Spiel:
Es klingt kompliziert, aber wir Vielnachrichtenempfänger brauchen irgendeine Hilfe zum Verwalten.
Nachteile für das Spiel:
Zu kompliziert.
Mögliche Problematiken:
Zu kompliziert.
