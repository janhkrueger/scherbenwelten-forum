So ist es momentan:
im gesamten Forum steht der ig-nick vor einem Beitrag, obwohl das Forum klar in rpg-Bereich und ooc-Bereich getrennt ist
So soll es werden:
ooc-statements im Forum werden nur durch einen extra anzugebenden ooc-nick gekennzeichnet, so dass kein Zusammenhang zum ig-Spieler erkennbar ist
Vorteile für das Spiel:
weniger Vermischung von ingame-Fakten mit dem Spieler dahinter, weil niemand weiß welcher ig-nick hinter dem ooc-Kommentar steckt
Nachteile für das Spiel:
für mich nicht erkennbar
Mögliche Problematiken:
eventuell mehr Chaos im Forum, weil einige wirklich nicht ooc von ig trennen können
