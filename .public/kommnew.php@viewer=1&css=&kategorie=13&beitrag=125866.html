Kampf/Armeen/Monster - Erweiterung von Kampfergebnissen (1 )
Chevonne Siobhan (RIP) So ist es momentan:
Im Kampf mit Armeen gibt es als Ergebnis nur den Tod von Soldaten. Gleiches gilt für Städte.
So soll es werden:
Das Kampfergebnis wird in verschiedene Kategorien eingeteilt.
1. Tote - werden, wie gehabt, von der Anzahl der Einheiten abgezogen.
2. Verletzte - werden vorerst von der Anzahl der Einheiten abgezogen. Sollte die Armee nicht völlig vernichtet werden, können sie von Heilern ab einem Skill X mit Y Kräutern gesund gepflegt werden und stehen nach Z Tagen wieder zur Verfügung.
3. Deserteure - werden von der Anzahl der Einheiten abgezogen. Je nach Skill Menschenkenntnis des Eigners kann ein gewisser Prozentsatz wieder eingefangen werden. Sie erscheinen dann als Sklaven im Besitz des Eigners.
4. Gefangene - von der Anzahl der Einheiten abgezogen. Sie stehen dann dem Gegner als Sklaven zur Verfügung.

Gleiches könnte man auch für Angriffe auf Städte verwenden. Sklaven würden dann auf dem Marktplatz erscheinen, so die Stadt den Sieg davonträgt.

Wie dann mit den Sklaven umgegangen wird, hängt natürlich von jedem selber ab.
Vorteile für das Spiel:
Eine neue Möglichkeit Sklaven zu erwerben. Eine Möglichkeit den Heiler noch interessanter zu gestalten.
Nachteile für das Spiel:
Ich erkenne derzeit keine.
Mögliche Problematiken:
Armeen werden "haltbarer". Sehe ich aber nicht unbedingt als Nachteil.
Sonstiges:
Ich hab mal ein wenig in den Schlachten unserer Geschichte geschaut. Für die letzten 200 Jahre kann man auf recht genaue Zahlen zurückgreifen. Daraus kann man sicherlich eine vernünftige Aufteilung in die 4 Kategorien ermitteln. 		04.09.10 23:36
