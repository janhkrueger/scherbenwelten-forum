Thomas Eisenhart
Folgende Idee kam mir im Zusammenhang mit einem anderen Thread, der aber inzwischen so lang ist, dass vermutlich kaum mehr einer dort mitliest...

Das Problem:

Es ist derzeit ohne Schwierigkeiten möglich, einen Schnupperaccount anzulegen, mit dem ein wenig Ruhm zu machen und anschließend mehrere Türme zu bauen, um damit ein Gebiet abzusperren. Ob man selbst den Schnupperaccount anlegt oder einen Kumpel findet, der das für einen tut, ist eigentlich egal. Auf jeden Fall kann man sich so erhebliche Vorteile verschaffen, ohne das Hindernis zu haben, dafür einen weiteren Account finanzieren zu müssen. (Gegen Leute, die das tun, hilft mein Ansatz natürlich nicht!)

Mein Lösungsvorschlag:

Türme von Schnupperaccounts können nur für Leute gesperrt werden, die man auf persönlicher KE hat. (Die Beschränkung auf persönliche KE beugt "billigen" Nat-KEs vor.) Darüber hinaus wird die 10%-Regel auf KEs ausgeweitet, sofern sie das nicht schon ist. (Ansonsten könnte man theoretisch 72 KEs pro Tag raushauen. Weiß zwar nicht, ob das jemand versuchen würde, aber Vorbeugen kann ja nicht schaden...)

Die Schnupperaccounts werden mit der Einschränkung wohl leben können und den Rest betrifft es nicht. 		24.08.04 15:41
