Kurze Einleitung:

Die Spielwelt ist aufgeteilt in diverse Städte. Jede Stadt hat mehrere Distrikte. Ein Spieler kontrolliert einen oder mehrere Distrikte.

Der Spieler wird im Spiel durch einen virtuellen Charakter repräsentiert, den er selbst gestalten kann. Je weiter er diesen OG aufrüstet, umso angesehener ist er und umso teurer kann er Drogen verkaufen und umso besser laufen seine Clubs.



Das Ansehen ist das Hauptziel des Spieles. Alle Bemühungen dienen letztendlich diesem Ziel. Je mehr Distrikte der User kontrolliert, je mehr und coolere Rides der User hat, je mehr Kämpfe sein OG und seine Homies überlebt haben umso höher steigt sein Ansehen.



So kann sich der User seine eigene Gangster-Legende bauen.
