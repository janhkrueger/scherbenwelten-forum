Magie - Heiraten und Titel (2 )
Rian Almaren
So ist es momentan:
Wenn man heiratet wird als Titel der jeweilige aktuelle Name des Partners gesetzt. Nun ist es bei einer Heirat aber recht häufig so das auch eine Umbenennung erfolgt - dies ist im Titel jedoch nicht berücksichtigt.
So soll es werden:
Bei einer Umbenennung sollte geprüft werden ob die Person verlobt oder verheiratet ist und wenn es so ist dann der Titel des Partners automatisch korrigiert werden.
Vorteile für das Spiel:
Eine kleine ´Unschönheit´ in der Signatur wird behoben und die Supporter müssen das nicht manuell machen.
