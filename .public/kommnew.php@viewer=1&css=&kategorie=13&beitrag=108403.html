Geist von Gabilkhazâd (RIP) So ist es momentan:
Eine Skillabfrage im Geredequest ist derzeit nicht möglich.
So soll es werden:
Eine Skillabfrage in der Form

hasskill=10&skillid=1
hasskillmax=10&skillid=1

soll möglich sein. Skillids sollte es schon geben und eine Liste der Skills könnte in der bekannten Hilfe mit veröffentlicht werden.
Vorteile für das Spiel:
Viele neue Möglichkeiten beim Questbauen.
Nachteile für das Spiel:
Für mich keine ersichtlich.
Mögliche Problematiken:
Die üblichen Verdächtigen:
- Programmieraufwand
- Intesives Testen notwendig, damit dieneue Funktion nicht unberechtigt ausgenutzt werden kann
