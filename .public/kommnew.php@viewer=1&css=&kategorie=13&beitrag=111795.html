Kimsa Tirjolfsson (RIP) So ist es momentan:
Die Lagerstädte haben kaum eine bedeutung.
So soll es werden:
Die Lagerstädte bieten sonderwaren (Thaum, mahagonie etc) an abhängig von ihrer größe.
Wieviel sie jedem einzelnen Spieler dabei verkaufen hängt von dessen Einfluss ab.
auch bekommen sie so geld um Waren einzukaufen und somit zu wachsen.
Vorteile für das Spiel:
Die Lagerstädte bekommen wieder etwas bedeutung. auch junge Spieler oder die die nicht so gute Kontakte zu großen Nationen haben, erhalten die möglichkeit an Sonderwaren für die eigene Produktion zu bekommen
Nachteile für das Spiel:
Balancing, Warenüberschuss bzw zu starker Abfluss von Massenwaren.
