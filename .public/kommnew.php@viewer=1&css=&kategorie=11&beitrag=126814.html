Iga Pelogran (RIP) Im Flash Links einzubauen ist eigentlich recht einfach.

1. Markiere das Objekt (z.B. Bild, Form, Text), das zum Link werden soll. Hast Du kein eigenes Objekt und möchtest einen Bereich als Link nutzen, musst Du einen durchsichtigen Kasten erstellen!
2.Drücke F8 um das Objekt in ein "Symbol" zu wandeln
3. Wähle Schlatfläche als Typ aus (und vergib idealerweise einen Namen für diese).
4. Drücke F9 für das Aktionen-Fenster
5. Wähle Symbol (Name) auf der linken Seite.
6. In der rechten Seite gibts Du den Quelltext ein:

on(release){
getURL("start.html","_blank");
}

Dies würde ein neues Fenster öffnen mit der Seite. Möchtest Du anstatt einem neuen Fenster das gleiche, nimm anstatt "_blank" "self".
