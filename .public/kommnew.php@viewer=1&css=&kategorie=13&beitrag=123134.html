So ist es momentan:
Man muss, will man eine Stadt erobern, sie entweder abreißen oder über mehrere Tage/Wochen belagern und aushungern
So soll es werden:
Steht eine Stadtbevölkerung einer doppelten Übermacht gegenüber (1Amreesoldat gilt als 50 Bewohner) stürmt die Bevölkerung, durch massive Panik, aus der Stadt und das Bürgermeisteramt wird für vakant erklärt und Umgehend Neuwahlen ausgerufen.
Das hieße, bei einer 100k Einwohnerstadt müssten 4000 Soldaten die Stadt belagern, um umgehend Neuwahlen auszurufen
Vorteile für das Spiel:
Müßige Belagerungen werden massiv verkürzt, indem man einfach eine entsprechend große Menge Armeen aushebt. So können auch kleine Städte von kleineren Nationen oder Spielergruppierungen erobert werden und so auch, ohne viele eigene PP, anderen Nationen erheblichen Schaden zuzufügen. Auch größere Städte sind bei guter Organisation in einem Husarenritt einzunehmen
